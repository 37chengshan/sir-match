# AGENTS.md

## 项目定位

这是电赛备赛仓库。当前重点是顶层题目文件夹：

```text
G-circuit-model/
```

该题为 G题「电路模型探究装置」，平台统一为 **MSPM0G3507**。

## 查找顺序

1. `README.md`：仓库总结构。
2. `比赛文档/00-总要求.md`：通用要求。
3. `比赛文档/01-MSPM0G3507训练计划.md`：当前训练计划。
4. `G-circuit-model/README.md`：G题入口。
5. `common/`：通用算法和硬件模块。

## 文件归档规则

- 题目文件夹必须在仓库顶层。
- 通用内容放 `common/`。
- 比赛总要求和训练计划放 `比赛文档/`。
- 不再使用 `problems/` 二级题目目录。
- 不再使用 `agent/` 目录。
- 不再保留旧 `stm32/` 和旧根目录方案文件作为主线。
