## 当前硬件模块

- MSPM0G3507 主控板；
- AD9833 DDS 信号源；
- 数字电位器幅度控制；
- 运放缓冲与偏置；
- Vin/Vout 双路 ADC 前端；
- DAC 波形复现输出；
- 已知 RC 模型电路；
- 未知 RLC 黑盒电路；
- OLED/ST7789 显示。

## 接线原则

- ADC 输入必须做 1.65V 偏置和限幅保护；
- 模拟地和数字地先共地，但模拟前端布线尽量短；
- AD9833 输出后必须缓冲，不直接硬拖模型电路；
- Vin 和 Vout 测试点必须预留，方便接示波器；
- 电源入口加开关、保险或限流保护。
