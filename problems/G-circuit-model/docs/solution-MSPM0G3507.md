# G题：电路模型探究装置 - MSPM0G3507 结构化方案

## 1. 方案结论

本题采用 **MSPM0G3507 + AD9833 + 双 ADC 同步采样 + DAC 波形复现** 的方案。

核心链路：

```text
AD9833 产生测试信号
    ↓
已知/未知模型电路
    ↓
MSPM0G3507 双 ADC 同步采 Vin/Vout
    ↓
Goertzel 提取幅值和相位
    ↓
建立频率响应模型 H(jω)
    ↓
DAC 查表输出复现波形
```

## 2. 为什么用 MSPM0G3507

原方案使用 STM32F407VET6，性能足够，但不贴近 TI 杯指定平台。改为 MSPM0G3507 后，训练内容可以直接迁移到 TI 平台。

| 项目 | 选择理由 |
|---|---|
| 主控 | MSPM0G3507 属于 TI MSPM0 系列，适合 TI 杯训练 |
| ADC | 双 12-bit 高速 ADC，适合 Vin/Vout 同步采样 |
| DAC | 内置 12-bit DAC，可做查表波形复现 |
| 外设 | SPI、UART、I2C、DMA、定时器资源够用 |
| 生态 | CCS Theia + MSPM0 SDK + SysConfig |

## 3. 硬件系统

### 3.1 总体框图

```text
┌─────────────────────────────────────────────────────────────┐
│ MSPM0G3507                                                  │
│                                                             │
│ SPI0 ─────► AD9833 DDS ─► 幅度控制 ─► 运放缓冲 ─► 被测电路IN │
│ ADC0 ◄────────────────────────────── 被测电路IN / Vin       │
│ ADC1 ◄────────────────────────────── 被测电路OUT / Vout     │
│ DAC0 ─────► RC重构滤波 ─► 运放缓冲 ─► 波形复现输出           │
│ I2C/SPI ──► OLED / ST7789 显示                              │
│ UART ─────► 串口导出 CSV                                    │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 信号源

- 使用 AD9833；
- 由 MSPM0G3507 通过 SPI 控制；
- 频率覆盖 1kHz~50kHz；
- 输出后接运放缓冲；
- 幅度由数字电位器或可变增益放大器控制。

### 3.3 采样前端

ADC 输入不能直接接正负摆幅信号，必须偏置到 1.65V。

推荐链路：

```text
Vin/Vout → 限流电阻 → 1.65V偏置 → 运放跟随 → RC抗混叠 → ADC
```

要求：

- 输入 2Vpp 信号映射到约 0.65V~2.65V；
- ADC 前加保护电阻和钳位二极管；
- Vin/Vout 尽量同步采样；
- 相位计算必须用 `phase_out - phase_in`。

### 3.4 已知模型电路

训练用已知模型：

```text
H(s) = 5 / ((2s + 1)(s + 1))
```

推荐材料：

```text
R1 = 20kΩ
RF = 20kΩ
R2 = 10kΩ
R3 = 5kΩ
C1 = 0.1µF
C2 = 0.1µF
```

### 3.5 未知黑盒电路

训练用 RLC 黑盒：

```text
R = 2kΩ
L = 10mH
C = 10nF
频率范围：1kHz~50kHz
输入：2Vpp 正弦、矩形及其他周期信号
```

## 4. 软件系统

### 4.1 状态机

```text
CALIBRATE → IDLE → BASIC_OUTPUT → SWEEP → LEARN → REPLICATE → VERIFY
```

| 状态 | 功能 |
|---|---|
| CALIBRATE | ADC零点、DAC中点、1.65V偏置校准 |
| IDLE | 等待按键或串口命令 |
| BASIC_OUTPUT | AD9833输出正弦，自动调幅 |
| SWEEP | 扫频采样，生成幅相数据 |
| LEARN | 识别滤波类型并拟合参数 |
| REPLICATE | DAC输出复现波形 |
| VERIFY | 误差评估和串口导出 |

### 4.2 频响测量

每个频点：

1. 设置 AD9833 频率；
2. 等待稳定；
3. ADC 同步采 Vin/Vout；
4. 对 Vin 做 Goertzel；
5. 对 Vout 做 Goertzel；
6. 计算增益 `|H| = |Vout| / |Vin|`；
7. 计算相位 `φ = φout - φin`；
8. 保存数据。

### 4.3 模型识别

| 类型 | 判据 |
|---|---|
| 低通 | 低频增益高，高频衰减 |
| 高通 | 低频衰减，高频增益高 |
| 带通 | 中间频率有峰值 |
| 带阻 | 中间频率有谷值 |

### 4.4 波形复现

优先级：

1. 正弦波复现；
2. 矩形波前 5~10 个谐波复现；
3. 任意周期波采样表复现；
4. m序列互相关估计冲激响应，做高分拓展。

## 6. 调试路线

1. MSPM0G3507 点灯、串口、OLED；
2. SPI 驱动 AD9833 输出固定频率；
3. ADC 单通道测 Vpp；
4. 搭建已知 RC 模型；
5. 数字电位器闭环调幅；
6. ADC 双通道同步采 Vin/Vout；
7. Goertzel 计算幅相；
8. 扫频导出 CSV；
9. MATLAB 验证 Bode 曲线；
10. DAC 查表输出复现波形。

## 7. 最小验收标准

完成以下内容才算第一版可用：

- AD9833 可输出 1kHz~50kHz 正弦；
- ADC 能测出 Vpp；
- 自动调幅能稳定到 2Vpp 附近；
- Vin/Vout 双通道能计算增益和相位；
- 对已知 RC 电路扫频曲线趋势正确；
- 串口能导出 CSV；
- DAC 能输出查表正弦。
