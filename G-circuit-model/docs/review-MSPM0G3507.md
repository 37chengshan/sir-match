# G题 MSPM0G3507 方案多维度审查

## 0. 总结结论

当前方案方向正确：

```text
MSPM0G3507 + AD9833 + Vin/Vout 双ADC同步采样 + Goertzel 幅相测量 + DAC 波形复现
```

它比原 STM32F407 方案更贴近 TI 杯平台要求，也更适合提前沉淀 TI MSPM0 的外设、采样、控制和报告能力。

但当前方案仍处在“方案可行，工程未落地”的阶段。真正高分的关键不在继续增加复杂算法，而在于：

1. 先跑通 AD9833 输出和 ADC 采样；
2. 再跑通 Vin/Vout 双通道幅相测量；
3. 再做自动幅度控制；
4. 最后做波形复现和 m序列冲高分。

|---|---|---|---|

评分：**8/10**。平台方向正确，但必须尽快从文档进入 MSPM0 实测。

## 2. 硬件方案审查

### 2.1 信号源

AD9833 做 1kHz~50kHz 扫频信号源是合理的。它比 MSPM0 DAC 直接扫频更稳定，也更容易覆盖题目频段。

主要风险：

- AD9833 模块输出幅度较小；
- 高频输出有镜像和幅度变化；
- 输出直接带负载会失真。

改进建议：

```text
AD9833 → RC滤波 → 运放缓冲 → 数字电位器/可变增益 → 被测电路
```

### 2.2 ADC 前端

方案中已经明确 1.65V 偏置和 Vin/Vout 双采样，这是正确的。

主要风险：

- ADC 输入保护不足；
- 运放不是轨到轨，导致削顶；
- 高频段前端衰减被误认为被测电路衰减；
- 只在面包板上搭高频模拟前端，寄生参数偏大。

建议：

- 每路 ADC 前串 1k~10k；
- 加钳位保护；
- 高频路径短；
- 先面包板验证，后续焊洞洞板或 PCB。

### 2.3 DAC 输出

MSPM0G3507 内置 DAC 做波形复现可行，但不要期待完美矩形波。

建议优先级：

1. DAC 输出 1kHz 正弦；
2. DAC 输出带相位偏移的正弦；
3. DAC 输出前 5~10 次谐波合成矩形波；
4. m序列/冲激响应复现。

评分：**7.5/10**。框架合理，但模拟前端和波形复现需要实测验证。

## 3. 算法方案审查

| 算法 | 结论 | 风险 | 建议 |
|---|---|---|---|
| Goertzel | 最适合扫频单频幅相 | 采样率/目标频率不一致会错 | 固定采样率，记录每个频点参数 |
| PID调幅 | 可行 | 数字电位器分辨率有限，容易抖动 | 第一版用分段逼近，第二版再 PID |
| 滤波识别 | 规则法足够第一版 | 扫频点少会跳过峰值/谷值 | 快扫后局部细扫 |
| RLC拟合 | 可以作为报告亮点 | L/C 误差和寄生参数影响大 | 用实测 L/C 做理论对照 |
| FFT/谐波 | 适合矩形波复现 | 资源占用较大 | 先用解析谐波，不做大 FFT |
| m序列互相关 | 冲高分方向 | 工程复杂，时间风险高 | 放到最后，不影响基本分 |

评分：**8/10**。算法路线对，但必须坚持“先 Goertzel，后 FFT/m序列”。

## 4. 得分策略审查

建议目标从“追求极限高分”改成“三层得分”：

| 层级 | 目标 | 优先级 |
|---|---|---|
| 保底分 | 正弦输出、自动幅度、已知模型测试、报告完整 | P0 |
| 稳定高分 | Vin/Vout 扫频、Bode 曲线、滤波类型识别、RLC 参数估计 | P1 |
| 冲高分 | DAC 波形复现、矩形波谐波、m序列时域辨识 | P2 |

不要让 P2 功能拖垮 P0/P1。

评分：**8.5/10**。分层正确，但需要把“验收门槛”写进开发节奏。

## 5. 采购清单审查

第一批采购基本合理，但建议补充几个关键小料：

| 补充器材 | 原因 |
|---|---|
| BAT54S/BAV99 钳位二极管 | ADC 输入保护 |
| 3296 多圈电位器 | 初期手动调幅和校准 |
| TL431/REF3025 | 参考电压和偏置校准 |
| BNC/SMA 接头、测试钩 | 示波器调试方便 |
| 洞洞板 | 面包板跑通后固化模拟前端 |
| 逻辑分析仪 | 调 SPI/I2C/UART 很省时间 |

评分：**7.5/10**。核心器材齐，但测试连接和保护器件需要补全。

## 7. 当前最大风险排序

| 排名 | 风险 | 严重程度 | 处理方式 |
|---:|---|---|---|
| 1 | MSPM0 工程未真正跑通 | 高 | 先建最小工程，完成点灯/串口/SPI/ADC |
| 2 | ADC 前端失真或相位不准 | 高 | 示波器验证 Vin/Vout，做同相信号相位校准 |
| 3 | 自动幅度控制不稳定 | 中高 | 先分段逼近，后 PID |
| 4 | DAC 复现波形失真 | 中高 | 先正弦，后矩形谐波 |
| 5 | m序列过早投入 | 中 | 放到 P2，不影响基本分 |
| 6 | 面包板高频寄生 | 中 | 核心模拟链路焊洞洞板/PCB |

## 9. 最终审查结论

方案总体评级：**A-**。

优势：

- 平台选择正确；
- 系统链路完整；
- Goertzel 路线适合 MSPM0；
- 仓库结构已清晰；
- 采购和训练计划具备可执行性。

短板：

- 固件工程还没真正建立；
- 模拟前端需要实测；
- DAC 复现能力不能过度承诺；
- m序列属于冲高分，不应成为主线阻塞项。

一句话结论：

**现在不要再扩展新概念，先把 MSPM0G3507 + AD9833 + ADC + OLED 的最小闭环跑通，跑通后再进入扫频建模和波形复现。**
