# TI杯电赛 G题：电路模型探究装置 - MSPM0G3507 指定平台方案

> 改版目标：把原 STM32F407VET6 方案调整为 **TI 比赛指定/推荐平台：MSPM0G3507**。  
> 核心原则：训练阶段尽量贴近现场器件；主控、ADC、DAC、SPI、DMA、数学运算全部围绕 MSPM0G3507 重新组织。  
> 当前只修改方案文档与 HTML 展示页，固件代码后续再按本方案迁移。

---

## 0. 改版结论

原方案基于 STM32F407VET6，性能宽裕，但与 2026 TI 杯指定平台不一致。改版后采用：

- **主控板：MSPM0G3507 核心板 / LP-MSPM0G3507 LaunchPad**
- **开发环境：Code Composer Studio Theia + MSPM0 SDK + SysConfig**
- **信号源：AD9833 DDS，SPI 控制输出 1kHz~50kHz 正弦测试信号**
- **采集：MSPM0G3507 双 ADC 同步采 Vin/Vout，DMA 缓冲**
- **输出复现：MSPM0G3507 内置 12-bit DAC + 运放缓冲，必要时叠加 AD9833 正弦输出**
- **建模算法：Goertzel 幅相提取为主，FFT/互相关/m序列为发挥验证方案**
- **显示交互：ST7789 TFT 或 0.96 寸 OLED，按键/旋钮完成模式切换**

**优先得分路径：先稳定完成基本要求和频响建模，再做波形复刻。不要一开始追求复杂 FFT 大屏动画。**

## 1. 依据与题目理解

### 1.1 平台依据

2026 年 TI 杯决赛现场提供的处理器平台包括 **MSPM0G3507 核心板、MSPM0L1306 核心板、MSP-EXP430F5529LP、LAUNCHXL-F280049C**。初赛至少有一道题会指定使用 TI MSPM0 系列处理器，型号不限定。因此训练方案选择 **MSPM0G3507** 作为主平台。

选择 MSPM0G3507 的原因：

| 项目 | MSPM0G3507 价值 |
|---|---|
| 主频 | 80MHz Cortex-M0+，足够跑 Goertzel、扫频状态机、显示刷新 |
| 存储 | 128KB Flash + 32KB SRAM，适合中小型裸机工程 |
| ADC | 双 12-bit 4MSPS ADC，适合同步采 Vin/Vout |
| DAC | 1 路 12-bit 1MSPS DAC，可做波形复现输出 |
| 外设 | 2 SPI、4 UART、DMA、定时器、MATH 加速器，适合电赛系统集成 |
| 模拟资源 | 内置 OPA、比较器、VREF，可减少部分外围模拟电路压力 |

### 1.2 G题任务理解

G题“电路模型探究装置”的本质不是单纯做信号发生器，而是做一个小型 **电路数字孪生系统**：

1. 对已知模型电路建立数学模型；
2. 根据已知模型设计 RC 有源滤波电路；
3. 对未知黑盒电路进行学习建模；
4. 根据学习得到的频率响应或冲激响应复现未知电路输出。

题目解析中给出的已知电路模型可写为：

```text
H(s) = 5 / ((2s + 1)(s + 1))
```

未知模型示例为 RLC 电路：

```text
R = 2kΩ
L = 10mH
C = 10nF
频率范围：1kHz ~ 50kHz，步长 200Hz
输入信号：2Vpp 正弦波、矩形波及其他周期信号
```

### 2.2 功能分层

| 层级 | 模块 | 职责 |
|---|---|---|
| 硬件驱动层 | AD9833、ADC、DAC、TFT/OLED、按键、串口 | 外设初始化与数据搬运 |
| 信号采集层 | 双 ADC 同步采样、DMA 双缓冲、零点校准 | 获得 Vin/Vout 波形 |
| 幅相测量层 | Goertzel、过零检测、幅值/相位差计算 | 得到单频点频率响应 |
| 建模识别层 | 低通/高通/带通/带阻识别、RLC 参数估计 | 得到 H(jω) 或 h(t) |
| 输出复现层 | DAC 查表输出、谐波补偿、逆滤波限幅 | 复现未知电路输出 |
| 交互显示层 | Bode 曲线、参数结果、模式状态 | 现场可测、可解释 |

### 2.3 主状态机

```text
CALIBRATE → IDLE → BASIC_OUTPUT → SWEEP → LEARN → REPLICATE → VERIFY
```

| 状态 | 功能 | 目标 |
|---|---|---|
| CALIBRATE | ADC 零点、1.65V 偏置、DAC 中点校准 | 降低系统误差 |
| IDLE | 等待按键/串口指令 | 状态稳定 |
| BASIC_OUTPUT | AD9833 正弦输出 + 自动幅度调节 | 完成基本要求 |
| SWEEP | 1kHz~50kHz 扫频，步长 200Hz 或自适应对数扫频 | 获得幅频/相频曲线 |
| LEARN | 滤波类型识别 + 参数拟合 | 形成模型 |
| REPLICATE | DAC/谐波法输出复现波形 | 冲发挥分 |
| VERIFY | 自测误差、显示结果、导出串口数据 | 支撑报告 |

## 4. 硬件设计

### 4.1 信号生成链路

```text
MSPM0G3507 SPI → AD9833 DDS → 数字电位器/可变增益 → 运放缓冲 → 被测电路输入
```

设计要求：

- 输出频率覆盖 1kHz~50kHz；
- 正弦波峰峰值自动调节到 2Vpp；
- 频率点可按 200Hz 步长精扫，也可先用 30~50 点对数扫频快速定位；
- AD9833 输出后必须加低通滤波和缓冲，避免高频镜像和负载影响；
- 幅度控制优先使用数字电位器，其次使用可控增益运放。

### 4.2 采样前端

MSPM0G3507 ADC 只能采 0~3.3V 区间信号，因此外部正负摆幅信号必须先做偏置和限幅。

推荐链路：

```text
Vin/Vout → 10k限流 → 1.65V偏置叠加 → 运放电压跟随 → RC抗混叠 → ADC
```

关键点：

- ADC 输入中心偏置为 1.65V；
- 输入 2Vpp 信号映射到约 0.65V~2.65V；
- 每路 ADC 前串 1k~10k 保护电阻；
- 对地/3.3V 加肖特基钳位或低电容保护；
- Vin 与 Vout 尽量同时采样，否则相位误差会变大；
- Goertzel 计算时使用 `Vout/Vin` 的幅度比和相位差，而不是只看单路输出。

### 4.3 已知模型电路

推荐按题目解析搭建两级 RC 有源滤波电路，便于理论分析和报告书写。

```text
R1 = 20kΩ
RF = 20kΩ
R2 = 10kΩ
R3 = 5kΩ
C1 = 0.1µF
C2 = 0.1µF
```

对应模型：

```text
H(s) = 5 / ((2s + 1)(s + 1))
```

如果现场更强调 Sallen-Key 形式，也准备一套 Sallen-Key 二阶低通参数，使用可插拔电阻电容座快速换值。

### 4.4 未知模型测试电路

训练时自制未知黑盒，用拨码开关或接插件隐藏参数。

基础 RLC 训练模型：

```text
R = 2kΩ
L = 10mH
C = 10nF
输入：1kHz~50kHz，2Vpp
```

建议同时准备 4 类黑盒：

| 类型 | 训练目的 |
|---|---|
| RC 低通 | 基本建模和截止频率识别 |
| RC 高通 | 类型识别鲁棒性 |
| RLC 带通 | 中心频率、Q 值拟合 |
| RLC 带阻 | 相位突变和陷波识别 |

### 4.5 波形复现链路

```text
学习得到 H(jω) 或 h(t)
        ↓
目标输入 x(t) 分解为谐波/采样序列
        ↓
计算 y(t)=x(t)*h(t) 或 Y(jω)=X(jω)H(jω)
        ↓
MSPM0G3507 DAC 查表输出
        ↓
RC 重构滤波 + 运放缓冲
```

实现优先级：

1. **正弦波复现**：只需幅值和相位补偿，最稳；
2. **矩形波复现**：取前 5~10 个有效谐波，受 DAC 更新率和模拟带宽限制；
3. **任意周期波复现**：采样表 + IIR/FIR 或冲激响应卷积，作为冲高分方向。

## 5. 软件算法方案

### 5.1 单频点测量

每个频点执行：

1. AD9833 设置频率；
2. 延时等待电路稳定；
3. ADC 同步采 Vin/Vout；
4. 对 Vin 和 Vout 分别做 Goertzel；
5. 计算增益 `|H| = |Vout| / |Vin|`；
6. 计算相位 `φ = φout - φin`；
7. 存入频响数组。

### 5.2 扫频策略

训练阶段使用两级扫频：

| 阶段 | 点数 | 目的 |
|---|---:|---|
| 快扫 | 30~50 点对数扫频 | 快速判断滤波类型和大致中心频率 |
| 精扫 | 200Hz 步长或局部细扫 | 拟合参数、提高报告数据质量 |

### 5.3 模型识别

| 类型 | 判据 |
|---|---|
| 低通 | 低频增益高，高频衰减，-3dB 点可定位 |
| 高通 | 低频衰减，高频增益高 |
| 带通 | 中间频率有峰值，左右两侧衰减 |
| 带阻 | 中间频率有谷值，相位附近跳变 |

### 5.4 参数拟合

基本路线：

- RC 低通/高通：由 -3dB 截止频率反推 RC；
- RLC 带通/带阻：由中心频率 `f0` 和带宽 `BW` 估计 `Q=f0/BW`；
- 进阶：使用最小二乘拟合 H(s) 参数；
- 报告中同时给出理论值、测量值、误差百分比。

### 5.5 冲激响应与 m 序列方向

发挥部分可保留时域辨识路线：

- DAC 输出 m 序列或伪随机二值序列；
- ADC 采输入 x(n) 和输出 y(n)；
- 通过互相关估计冲激响应 h(n)；
- 再用卷积 `y(n)=x(n)*h(n)` 进行数字孪生复现。

该方案更复杂，不作为第一阶段必做项，但适合冲击高分和报告创新点。

## 8. 实施路线

### 第 1 阶段：MSPM0G3507 基础外设

- 点灯、按键、串口打印；
- SPI 驱动 AD9833；
- I2C/OLED 或 SPI/ST7789；
- ADC 单通道采样；
- DAC 输出正弦查表；
- DMA 双缓冲。

### 第 2 阶段：基本要求闭环

- AD9833 输出 1kHz 正弦；
- ADC 采 Vout，计算 Vpp；
- 数字电位器调节幅度；
- PID 使输出稳定到 2Vpp；
- OLED/TFT 显示频率、Vpp、误差。

### 第 3 阶段：扫频建模

- 双通道同步采 Vin/Vout；
- 1kHz~50kHz 扫频；
- Goertzel 计算增益和相位；
- 识别低通/高通/带通/带阻；
- 串口导出 CSV，MATLAB 画图验证。

### 第 4 阶段：波形复现

- 先做正弦波幅相复现；
- 再做矩形波前 5~10 次谐波合成；
- 最后尝试 m 序列互相关得到 h(n)，用卷积复现。

## 9. 关键风险与规避

| 风险 | 表现 | 规避 |
|---|---|---|
| MSPM0 资源少于 F407 | 大 FFT、复杂 UI 卡顿 | 用 Goertzel、分段处理、减少浮点数组 |
| ADC 相位误差 | Bode 相频不准 | Vin/Vout 同步采样，固定采样窗口 |
| AD9833 幅度不可控 | 2Vpp 不稳定 | 数字电位器 + 运放缓冲 + PID |
| 运放单电源失真 | 波形削顶 | 1.65V 偏置，选 RRIO 运放，必要时用 ±5V |
| 面包板寄生参数 | 高频段误差大 | 高频路径短、加地线、最终焊洞洞板/PCB |
| 复现波形高频谐波丢失 | 矩形波边沿差 | 限制谐波数，报告中说明带宽限制 |
| 下载器误用 | MSPM0 芯片异常 | 不用 ST-LINK，使用 XDS110/官方方式 |

## 10. 最小可运行验收标准

项目真正进入可比赛状态，至少要满足：

1. MSPM0G3507 能稳定控制 AD9833 输出 1kHz~50kHz；
2. 输出正弦经被测电路后，ADC 能正确测量 Vpp；
3. 自动幅度控制能把输出稳定到 2Vpp 附近；
4. Vin/Vout 双通道采样能算出增益和相位差；
5. 对已知 RC 电路扫频，Bode 曲线与理论趋势一致；
6. 对 RLC 黑盒能识别中心频率和 Q 值；
7. DAC 能输出查表正弦/矩形近似波形；
8. 串口能导出测试数据用于报告。
