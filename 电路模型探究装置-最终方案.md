## 二、方案演进历程

### 2.1 三轮审核改进

| 轮次 | 核心变化 | 原因 |
|:---:|:---|:---|
| 第1轮 | F103→F401, ADS1115→内部ADC | 外部ADC采样率瓶颈(860SPS) |
| 第2轮 | 增加Goertzel+相位识别, PID自动微调 | 提升建模精度和速度 |
| **第3轮** | **F401→F407, OLED→ST7789, 增加DAC** | **DAC是波形复刻的最佳路径** |

### 2.2 MCU选型最终对比

| 参数 | F401CCU6 | **F407VET6 (选定)** |
|:---|:---:|:---:|
| 主频 | 84MHz | **168MHz** |
| FPU | ✓ | ✓ |
| SRAM | 64KB | **192KB** |
| ADC | 1×2.4MSPS | **3×2.4MSPS** |
| **DAC** | ❌ 无 | **✅ 2×12-bit** |
| SPI | 3路 | **3路** |
| 价格 | ¥30 | **¥40** |

**关键决策理由**：波形复刻（40分）需要输出任意补偿波形。有DAC可以直接将数字补偿后的波形经DAC输出；无DAC只能用AD9833逐谐波合成，复杂且易出错。

### 3.3 已知模型电路

**传递函数**：$$H(s)=\frac{5}{10^{-8}s^2+3\times10^{-4}s+1}$$

**电路实现：两级RC低通 + 同相放大（增益=5）**

```
第一级 fc≈1592Hz      第二级 fc≈1561Hz      同相放大 K=5
 R1=10kΩ 1%           R2=15kΩ 1%           Rf=40kΩ 1%
Vin ─┤├─── V1 ─┤├─── V2 ──[OPA2134(+)]──┤├─┬── Vout
        │         │              │            │
       C1        C2              └── Rg=10kΩ──┘
      10nF 5%   6.8nF 5%             1%
        │         │              │
       GND       GND            GND
```

**参数验证（MATLAB仿真）**：
- fc1 = 1/(2π×10k×10n) = 1592 Hz
- fc2 = 1/(2π×15k×6.8n) = 1561 Hz
- 1kHz处增益 = 2.53 (8.05 dB)，所需输入峰峰值 = 0.79V (输出2Vpp)
- 预期总误差 < 8%（满足≤10%要求）

## 四、软件设计

### 4.1 软件架构

```
┌─────────────────────────────────────────────────┐
│              应用层 (main.c)                     │
│  状态机: IDLE→SWEEP→AUTO_AMP→LEARN→REPLICATE    │
├─────────────────────────────────────────────────┤
│  算法模块:  goertzel | pid | filter_id | rlc_fit│
│  输出模块:  waveform (DAC复刻)                   │
├─────────────────────────────────────────────────┤
│  驱动层:  ad9833 | adc_driver | dac_driver      │
│  显示层:  st7789                                 │
├─────────────────────────────────────────────────┤
│  HAL层:  SPI1/SPI2 | ADC1+DMA | DAC1+DMA | TIM2│
└─────────────────────────────────────────────────┘
```

### 4.2 核心算法

| 算法 | 用途 | 关键参数 | MATLAB验证 |
|:---|:---|:---|:---:|
| Goertzel | 单频率幅度+相位提取 | 512点, O(N) | 幅度误差<1% |
| PID | AD9833自动幅度控制 | Kp=0.5, Ki=100, Kd=0.001 | 稳态误差<5% |
| 滤波识别 | 幅度+相位联合判定 | 4类型, 5频点确认 | 100%准确率 |
| RLC拟合 | 解析估算+局部优化 | f0→L,C, Q→R | 参数误差<10% |
| 逆滤波+DAC | 波形复刻补偿输出 | FFT谐波分解+频域补偿 | RMSE<0.05 |

### 4.3 DAC波形复刻流程

```
输入信号 ─→ ADC采样 ─→ FFT谐波分析 ─→ 各谐波逆滤波补偿 ─→ IFFT合成 ─→ DAC输出
                                  ↑
                           RLC电路参数
                         (学习阶段获取)
```

## 六、风险控制

| 风险 | 概率 | 影响 | 对策 | 降级方案 |
|:---|:---:|:---:|:---|:---|
| F407 DAC精度不足 | 低 | 中 | 12-bit+3.3V参考, 分辨率0.8mV | 过采样+软件滤波 |
| ST7789 SPI干扰 | 低 | 低 | 独立SPI2总线, 不与AD9833共享 | 降低SPI时钟 |
| 矩形波复刻失真 | 中 | 中 | FFT谐波分解+DAC直接合成 | 降级为5次谐波近似 |
| 面包板50kHz噪声 | 高 | 低 | 屏蔽线+ADC端中值滤波 | 降低扫频上限 |
