# 比赛文档

这里放和整个电赛备赛相关的总文档，不放具体题目的专用方案。

## 当前文件

```text
比赛文档/
├── README.md
├── 00-总要求.md
└── 01-MSPM0G3507训练计划.md
```

## 应该放这里的内容

- 总要求；
- 训练计划；
- 采购原则；
- 官方规则摘要；
- 平台选型原则；
- 比赛时间安排；
- 报告通用模板；
- 通用调试流程。

## 不应该放这里的内容

- 某个题目的专用方案；
- 某个题目的固件；
- 某个题目的仿真脚本；
- 某个题目的测试数据。

这些应该放入对应顶层题目文件夹，例如 `G-circuit-model/`。
