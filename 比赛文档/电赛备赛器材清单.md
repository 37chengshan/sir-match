# 电赛备赛器材清单

> 来源：用户提供的器材截图与后续小车方案整理。价格为估算价，单位默认人民币元；部分闲鱼链接在截图中被截断，本文只保留可确认信息。  
> 当前统一口径：主控为 **地猛星 MSPM0G3507**，小车/云台视觉主线为 **立创·庐山派 K230 / CanMV K230**。

## 1. 结论版采购建议

### 1.1 优先买

| 类别 | 建议优先采购 | 理由 |
|---|---|---|
| 主控与下载 | 地猛星开发板、XDS110 仿真器 | 地猛星体积更小；XDS110 比 JLINK/ST-LINK 更适合 MSPM0/TI 开发流程。 |
| 基础搭建 | 杜邦线、面包板、插件包、USB 转 TTL、电源模块 | 每次调试都会用，且电源模块最容易损坏，建议备份。 |
| 小车方向 | 两轮差速小车、TB6612 模块、五路/八路灰度模块、锂电池与充电器 | 对控制题、小车循迹、编码器、PID 训练最直接。 |
| K230 视觉方向 | 庐山派 K230、TF 卡、TF 读卡器、摄像头支架、独立 5V 稳压 | 当前视觉主线；K230 没有 TF 卡基本不能完整练视觉。 |
| 云台方向 | 开环 42 步进电机二维云台、步进电机驱动模块 | 控制题常见训练对象，后续可扩展 K230 视觉对准。 |
| 显示与调试 | OLED 屏幕、万用表、逻辑分析仪 | OLED 用于现场状态显示；万用表/逻辑分析仪是基础排障工具。 |

### 1.2 数量建议

- 地猛星开发板：建议买 2 块。
- 电源模块：建议多买备份。
- 锂电池与充电器：建议买 2 套，轮换使用。
- USB 转 TTL：建议多买几个，CH340 系列一般可用。
- 杜邦线：公对公、母对母、公对母三种都买一些，2.54 mm 规格。
- K230 TF 卡：8GB/16GB Class 10/U1，建议至少 1 张，最好再备 1 张。

## 4. 采购注意事项

1. **主控优先选地猛星开发板**：体积小，最好买两个。
2. **下载调试优先 XDS110**：JLINK/ST-LINK 在 TI/MSPM0 生态里容易增加调试成本。
3. **电源模块要备份**：电源是整个系统最容易出故障的环节，建议至少准备 2 个。
4. **小车优先选择带霍尔编码器版本**：后续做速度闭环、里程计、PID 时更方便。
5. **K230 必须准备 TF 卡和读卡器**：推荐 8GB/16GB Class 10/U1，最低持续写入约 10MB/s。
6. **K230 要独立稳定供电**：建议 5V/2A 以上，最好单独一路降压，并与 MSPM0 共地。
7. **OpenMV 不再作为主线**：后续文档统一围绕 K230，避免代码和计划混乱。
8. **激光类模块要单独标记安全风险**：不要直视光束，也不要对人眼、镜面或强反射物测试。
9. **实验室已有的东西可不重复买**：插件包、示波器、部分线材和基础元件可以先确认实验室库存。
