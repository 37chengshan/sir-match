# MSPM0G3507 电赛训练计划

## 0. 当前结论

现在训练主线是：

```text
MSPM0G3507 → AD9833信号源 → 已知/未知模型电路 → 双ADC采样 → Goertzel建模 → DAC波形复现
```

目标不是一次做完所有高分功能，而是先把最小闭环跑通，再逐步升级。

---

| 类别 | 器材 | 数量 | 目标 |

| 模型电路 | 精密电阻、电容、电感套装 | 1批 | 搭建 RC/RLC 模型 |
| 连接 | 面包板、杜邦线、排针排母 | 1批 | 快速搭建 |

## 3. 第三阶段：基本要求闭环

### 3.1 最小闭环

```text
AD9833输出正弦 → 运放缓冲 → 已知模型电路 → ADC采样 → 计算Vpp → 数字电位器调幅
```

### 3.2 关键任务

1. 做 1.65V 偏置电路；
2. ADC 采被测输出 Vout；
3. 计算峰峰值 Vpp；
4. 通过数字电位器调节 AD9833 输出幅度；
5. PID 或分段逼近，使输出稳定到 2Vpp；
6. OLED 显示频率、Vpp、误差。

### 3.3 阶段验收

- 输入 1kHz 正弦时，输出 Vpp 能稳定在目标附近；
- 调整频率后，系统能重新稳定；
- 串口能打印：频率、目标Vpp、实测Vpp、误差、数字电位器值。

---

## 4. 第四阶段：扫频与频响建模

### 4.1 双通道采样

必须同时采：

- `Vin`：进入模型电路前的输入；
- `Vout`：模型电路输出。

不要只测 Vout。只测 Vout 会导致相位没有参考，无法得到真正的传递函数相位。

### 4.2 Goertzel 幅相测量

每个频点流程：

```text
设置AD9833频率
等待稳定
采Vin/Vout
对Vin做Goertzel
对Vout做Goertzel
增益 = |Vout| / |Vin|
相位差 = phase_out - phase_in
保存频率、增益、相位
```

### 4.3 扫频策略

- 快扫：30~50 个对数频点；
- 精扫：1kHz~50kHz，步长 200Hz 或局部细扫；
- 串口导出 CSV，MATLAB 画 Bode 图。

### 4.4 阶段验收

- 对 RC 低通电路，幅频曲线趋势正确；
- 对 RLC 带通电路，能找到中心频率；
- 串口导出数据可被 MATLAB 读取并画图。

---

## 5. 第五阶段：模型识别与参数拟合

### 5.1 类型识别

识别类型：

- 低通；
- 高通；
- 带通；
- 带阻。

判据：

- 低频高、高频低：低通；
- 低频低、高频高：高通；
- 中间峰值：带通；
- 中间谷值：带阻。

### 5.2 参数拟合

- RC 电路：用 -3dB 截止频率估算 RC；
- RLC 电路：用中心频率和带宽估算 Q；
- 报告中给出理论值、测量值、误差百分比。

### 5.3 阶段验收

- 至少能识别 4 类滤波器；
- 对训练黑盒能输出中心频率、Q 值和误差；
- OLED/TFT 能显示识别结果。

---

## 6. 第六阶段：波形复现

### 6.1 先做正弦波

正弦波最稳，优先实现：

```text
目标频率 f
学习得到 H(jw)
计算输出幅值和相位
DAC 输出补偿后的正弦表
```

### 6.2 再做矩形波

矩形波使用谐波法：

- 分解为基波 + 奇次谐波；
- 只取前 5~10 个有效谐波；
- 每个谐波乘以模型频响；
- 合成输出表；
- DAC 输出。

### 6.3 冲高分：m序列互相关

如果时间足够，做时域辨识：

```text
DAC输出m序列 → ADC采输入输出 → 互相关估计h(n) → 卷积复现y(n)
```

### 6.4 阶段验收

- DAC 可输出正弦复现波形；
- 矩形波复现趋势正确；
- 示波器截图能支撑报告。
