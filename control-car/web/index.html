📈 PID 控制参数

增量式 PI（速度环）

duty += kp * (err - last_err) + ki * err

kp = 0.5  ki = 0.4  控制周期: 1kHz (TIMA0)
PWM 周期: 4000 (10kHz)  起步速度: 125 mm/s

调参顺序

  1. 不循迹，先调左右速度 P
  2. 加少量 I 减少稳态误差
  3. 低速开环循迹（125 mm/s）
  4. 调循迹 Kp 回到线上
  5. 加 Kd 减少左右摇摆
  6. 逐步提高基础速度

⚠ 安全与避坑

  • MSPM0G3507 使用 XDS110 下载，不使用 ST-LINK
  • 电机/舵机不要从开发板直接取大电流，TB6612 VM 直连电池
  • 所有模块 GND 必须共地（电池→电源模块→TB6612→MSPM0）
  • TB6612 VM 附近并联 100~470μF 电解电容
  • 激光实验必须经过老师或实验室安全确认，调试阶段用普通 LED 替代
  • 小车底层不稳时不堆视觉/云台，先跑通阶段 1 再扩展
