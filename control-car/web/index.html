0
阶段 0：环境搭建

CCS Theia + XDS110 + LED → OLED → UART → 按键/ADC → 舵机。先确认 MSPM0 工具链稳定。

1
阶段 1：小车最小闭环

TB6612 双电机 PWM + 编码器测速 + 增量式 PI 速度闭环 + 5路/8路灰度循迹。

2
阶段 2：工程化

状态机 + UART 命令行调参 + OLED 多页状态显示 + CSV 测试数据记录 + 故障排查文档。

3
阶段 3：K230 视觉接入

TF 卡启动 CanMV → 摄像头预览 → 色块/黑线/二维码/AprilTag → UART 输出 VISION,...。

4
阶段 4：视觉 + 云台扩展

K230 输出目标坐标 → MSPM0 解析 → DCC-100v3 步进云台对准。视觉只做协处理，实时控制仍由 MSPM0 完成。

📈 PID 与视觉协议

增量式 PI（速度环）

duty += kp * (err - last_err) + ki * err

kp = 0.5  ki = 0.4
控制周期: 1kHz (TIMA0)
PWM 周期: 4000 (10kHz)
起步速度: 125 mm/s

⚠ 安全与避坑

  • MSPM0G3507 使用 XDS110 下载，不使用 ST-LINK
  • K230 需要 TF 卡完整练视觉，推荐 8GB/16GB Class 10/U1
  • K230 要独立 5V 稳压供电，所有模块 GND 必须共地
  • 电机/舵机不要从开发板直接取大电流，TB6612 VM 直连电池
  • TB6612 VM 附近并联 100~470μF 电解电容
  • K230 只做视觉协处理器，不做电机实时控制
  • 激光实验必须经过老师或实验室安全确认，调试阶段用普通 LED 替代
  • 小车底层不稳时不堆视觉/云台，先跑通阶段 1 再扩展
