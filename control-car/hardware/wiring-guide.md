# 接线教程：地猛星 MSPM0G3507 + TB6612 循迹小车

> 接线依据：`参考文件-已实现案例/2026_04_地猛星电赛控制题配套资料/` 中已验证的工程。
> 第一次测试时，小车轮子不要落地，先架空验证方向和 PWM。

## 1. 接线总原则

最小系统：

```
7.4V 锂电池
  ├─ 5V/3.3V 电源模块 → 地猛星开发板 + 灰度 + OLED
  └─ TB6612 VM（电机电源）

地猛星 MSPM0G3507
  ├─ TB6612（PWM + 方向 + STBY）
  ├─ 编码器（左右各 AB 相）
  ├─ 5路灰度传感器
  └─ OLED（I2C）
```

**所有 GND 必须共地**（电源模块、TB6612、编码器、灰度、OLED 的 GND 全部连在一起）。

## 2. 第一步：电源检查

连接前先用万用表检查：

| 测点 | 正常情况 |
|------|---------|
| 电池正负极 | 不短路 |
| 7.4V 与 GND | 约 6.6V~8.4V |
| 电源模块 5V | 约 5.0V |
| 电源模块 3.3V | 约 3.3V |

电源不正常不要插开发板和电机。

## 3. 第二步：接 TB6612 电机驱动

### 3.1 电源接线

| TB6612 | 连接 |
|--------|------|
| VM | 电池正极 7.4V（直接接电池，不走电源模块） |
| VCC | 3.3V（逻辑电，从电源模块取） |
| GND | 电源 GND（与电池、开发板共地） |

> VM 附近并联一个大电解电容（100~470μF），减少电机启动时的电压跌落。

### 3.2 左电机控制

| MSPM0 | TB6612 | 说明 |
|-------|--------|------|
| PA12 | PWMA | PWM 输出，TIMG0 CCP0 |
| PA9 | AIN1 | 方向控制 1 |
| PA8 | AIN2 | 方向控制 2 |
| PB24 | STBY | 待机/使能（高电平使能） |

### 3.3 右电机控制

| MSPM0 | TB6612 | 说明 |
|-------|--------|------|
| PA13 | PWMB | PWM 输出，TIMG0 CCP1 |
| PA7 | BIN1 | 方向控制 1 |
| PB18 | BIN2 | 方向控制 2 |

### 3.4 TB6612 输出 → 电机

| TB6612 | 左电机 | 右电机 |
|--------|--------|--------|
| AO1 | M+ | — |
| AO2 | M- | — |
| BO1 | — | M+ |
| BO2 | — | M- |

## 4. 第三步：接编码器

### 4.1 左电机编码器

| MSPM0 | 编码器 | 说明 |
|-------|--------|------|
| PA21 | A 相 | GPIO 中断计数 |
| PA22 | B 相 | GPIO 中断计数（判断方向） |
| 3.3V | VCC | 编码器供电 |
| GND | GND | 共地 |

### 4.2 右电机编码器

| MSPM0 | 编码器 | 说明 |
|-------|--------|------|
| PB19 | A 相 | GPIO 中断计数 |
| PB20 | B 相 | GPIO 中断计数（判断方向） |
| 3.3V | VCC | 编码器供电 |
| GND | GND | 共地 |

> 编码器供电电压以模块规格为准，常见 3.3V 或 5V。

## 5. 第四步：接灰度循迹传感器

5 路灰度模块，从左到右：

| 传感器 | MSPM0 | 说明 |
|--------|-------|------|
| L2（最左） | PA17 | GPIO 输入 |
| L1 | PB8 | GPIO 输入 |
| M（中间） | PB9 | GPIO 输入 |
| R1 | PA24 | GPIO 输入 |
| R2（最右） | PA2 | GPIO 输入 |

| 灰度模块电源 | 连接 |
|-------------|------|
| VCC | 按模块规格，3.3V 或 5V |
| GND | GND 共地 |

先不要跑车，用串口打印每一路黑白状态确认传感器方向：

```
S: 00100  表示中间传感器检测到黑线
S: 01100  黑线偏左
S: 00011  黑线偏右
```

## 6. 第五步：接 OLED

I2C OLED：

| OLED | 地猛星 |
|------|--------|
| VCC | 3.3V 或 5V（按模块规格） |
| GND | GND |
| SCL | I2C SCL |
| SDA | I2C SDA |

驱动：SSD1306 / SH1106，I2C 地址 0x3C 或 0x3D。

## 7. 第六步：串口调试

| USB 转串口 | 地猛星 |
|-----------|--------|
| GND | GND |
| TXD | RXD |
| RXD | TXD |

串口参数：**115200 8N1**。TX/RX 交叉，GND 必须相连。

## 8. 上电顺序

1. **不接电机电源（VM）**，只接逻辑电，测各点电压
2. 插 MSPM0，确认串口有启动日志
3. 接 OLED，确认有显示
4. 接一个电机，**架空**测试正反转和 PWM 调速
5. 接第二个电机，架空测试
6. 接编码器，手转轮子确认计数和方向
7. 接灰度模块，手动扫黑线确认读数
8. 最后接 VM 电机电源，**低速落地**测试

## 9. 常见问题

| 现象 | 可能原因 | 处理 |
|------|---------|------|
| 电机不转 | STBY 没拉高、VM 没供电、PWM=0 | 检查 PB24 输出高，万用表量 VM |
| 电机一转开发板复位 | 电源电流不足、VM 跌落拉低逻辑电 | VM 加大电容，检查电池电量 |
| 左右方向相反 | 电机接线或方向定义反了 | 软件里设方向系数 |
| 编码器计数为 0 | 中断没使能、供电不对、A/B 相接反 | 示波器看编码器波形 |
| 串口乱码 | 波特率不对、TX/RX 接反、没共地 | 115200 8N1，TX/RX 交叉，确认 GND |
| OLED 不亮 | 地址错、供电不对 | 检查 0x3C/0x3D，测 SCL/SDA |
| 灰度全 0 或全 1 | 供电、高度、阈值问题 | 单路测试，调整传感器高度 |
| 速度不稳 | 编码器干扰、PID 参数不对 | 先看编码器原始脉冲是否稳定 |
