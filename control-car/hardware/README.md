# 小车硬件工作区

本目录用于放置 MSPM0G3507 循迹小车的硬件资料：引脚表、接线图、电源设计、PCB 说明、模块安装方式和故障排查。

```text

bom.md             实际采购型号、链接、数量、价格、备用件
```

## 硬件原则

- 小车优先采用两轮差速 + 万向轮；
- 电机优先使用 N20 带编码器电机；
- 电机驱动优先使用 TB6612FNG；
- 电机电源和逻辑电源分清，但必须共地；
- 电机驱动附近加大电解电容；
- MSPM0G3507 不使用 ST-LINK 下载；
- 光源/激光相关实验必须经过老师或实验室安全确认。
