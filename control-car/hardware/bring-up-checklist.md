# 硬件 Bring-up 检查表

> 用途：每次焊完、改线、换模块后按表检查，避免一上电就烧模块或小车乱跑。

## 1. 空板/焊接检查

- [ ] P1/P2 排母方向正确；
- [ ] H2 顶部扩展口无连锡；
- [ ] 电池座正负极方向正确；
- [ ] D1 SS34 方向正确；
- [ ] SW1 开关能正常断开/闭合；
- [ ] U6 降压模块方向正确；
- [ ] R4/R5/R6 电池分压电阻为 10k；
- [ ] 蜂鸣器、LED 极性正确；
- [ ] 电机接口没有虚焊。

## 2. 不插主控时的电源检查

- [ ] 7.4V-GND 不短路；
- [ ] 5V-GND 不短路；
- [ ] 3V3-GND 不短路；
- [ ] 打开开关后 7.4V 正常；
- [ ] 打开开关后 5V 正常；
- [ ] ADC 约等于电池电压的 1/3。

## 3. 插主控后的检查

- [ ] MSPM0G3507 方向正确，没有错位；
- [ ] Type-C 能识别开发板；
- [ ] 不使用 ST-LINK 下载 MSPM0；
- [ ] 固件启动后默认电机输出为 0；
- [ ] 串口能输出 BOOT 日志；
- [ ] OLED 能显示 IDLE；
- [ ] START/MODE 按键原始值能打印。

## 4. 电机测试

每次测试前，小车轮子先架空。

- [ ] LQ 电机正转正常；
- [ ] LQ 电机反转正常；
- [ ] RQ 电机正转正常；
- [ ] RQ 电机反转正常；
- [ ] 左右轮“前进方向”已在软件中校准；
- [ ] PWM 从小到大速度变化正常；
- [ ] 电机启动时主控不复位；
- [ ] 电机停止命令可靠。

## 5. 循迹测试

- [ ] 循迹模块 VCC/GND 正确；
- [ ] 每一路 OUT 在白底/黑线下有变化；
- [ ] 黑线左偏时 error 为负；
- [ ] 黑线右偏时 error 为正；
- [ ] 黑线居中时 error 接近 0；
- [ ] 丢线时有保护逻辑。

## 6. 落地跑车前

- [ ] 电池固定；
- [ ] 主控固定；
- [ ] 电机线固定；
- [ ] 循迹传感器高度固定；
- [ ] OLED/串口显示正常；
- [ ] START 能停车；
- [ ] 最大 PWM 已限制；
- [ ] 场地黑线宽度和传感器高度合适。
