# 启动检查表：地猛星 + TB6612 循迹小车

> 按参考案例 `2026_04_地猛星电赛控制题配套资料/` 中 01→02→03 工程递进顺序编写。

## 1. 环境准备

- [ ] CCS Theia 已安装（`01_CCS_20.5.0.00028_win.zip`）
- [ ] XDS110 驱动已安装（`00_CH341SER.EXE`）
- [ ] 地猛星开发板引脚图已阅读（`00_立创·地猛星MSPM0G3507开发板引脚图.pdf`）
- [ ] 地猛星开发板原理图已阅读（`00_立创·地猛星MSPM0G3507开发板原理图.pdf`）
- [ ] 可创建、编译、下载空工程

## 2. 外设基础测试

对应参考案例：01_LED → 02_CLOCK_GPIO → 03_Debug → 05_OLED → 06_uart → 08_KEY_ADC → 09_PWM_SERVO

- [ ] LED 点亮/闪烁正常
- [ ] UART 串口打印正常（115200 8N1）
- [ ] OLED 显示正常（I2C）
- [ ] 按键读取正常
- [ ] ADC 读取正常（旋钮电位器）
- [ ] PWM 输出正常（舵机测试）

## 3. 电源检查

- [ ] 电池电压正常（6.6~8.4V）
- [ ] 电源模块 5V 输出正常
- [ ] 电源模块 3.3V 输出正常
- [ ] TB6612 VM 接电池正极
- [ ] TB6612 VCC 接 3.3V
- [ ] 各模块 GND 共地
- [ ] TB6612 VM 附近已并联电解电容（100~470μF）
- [ ] ADC 电池电压分压采样正常

## 4. 电机驱动测试（对应参考案例 01_DC_MOTOR_PID_1）

**架空测试（轮子不落地）：**

- [ ] TB6612 STBY 拉高（PB24 输出高电平）
- [ ] 左电机正转正常（PA9=1, PA8=0, PA12 PWM=小值）
- [ ] 左电机反转正常（PA9=0, PA8=1, PA12 PWM=小值）
- [ ] 右电机正转正常（PA7=1, PB18=0, PA13 PWM=小值）
- [ ] 右电机反转正常（PA7=0, PB18=1, PA13 PWM=小值）
- [ ] PWM 从低到高可明显调速
- [ ] 记录最小可转 PWM（死区）
- [ ] 确认左右轮"前进"方向，记录方向系数
- [ ] 电机停止正常（AIN1=AINT2=0）

## 5. 编码器测试（对应参考案例 02_DC_MOTOR_PID_2）

- [ ] 手转左轮，PA21 中断计数正常
- [ ] 手转左轮，PA22 中断计数正常
- [ ] 手转右轮，PB19 中断计数正常
- [ ] 手转右轮，PB20 中断计数正常
- [ ] 左右编码器计数方向与电机旋转方向一致
- [ ] 1kHz 定时器中断正常触发（TIMA0, LOAD_VALUE=39999）
- [ ] 速度计算数值合理（与手持转速表/目测比对）
- [ ] PWM 增大时速度数值单调增大
- [ ] 串口/OLED 可显示左右轮实时速度
- [ ] 电机死区已测量

## 6. 速度 PID 测试（对应参考案例 02_DC_MOTOR_PID_3）

- [ ] 左轮速度 PID 闭环稳定（kp=0.5, ki=0.4 起步）
- [ ] 右轮速度 PID 闭环稳定
- [ ] 目标速度改变后能平稳跟踪
- [ ] 无剧烈振荡
- [ ] 电池电压变化时速度仍相对稳定
- [ ] 串口可动态修改 kp/ki/kd
- [ ] 左右轮同目标速度时小车直线偏航可接受

## 7. 灰度传感器测试

- [ ] 5 路传感器供电正常
- [ ] 手动扫黑线，各路电平变化正确（白底=0, 黑线=1 或反之，确认极性）
- [ ] 串口打印传感器状态：`S: 00100` 格式
- [ ] 黑白阈值合理（不会频繁跳变）
- [ ] 确认传感器物理顺序与代码索引一致

## 8. 循迹测试（对应参考案例 03_PID_car）

**第一阶段：开环低速**

- [ ] 低速（125 mm/s）开环循迹可跑整圈
- [ ] 黑线左偏时 line_error 为负
- [ ] 黑线右偏时 line_error 为正
- [ ] 中间压线时 line_error 接近 0
- [ ] 丢线时进入减速/搜索状态

**第二阶段：闭环循迹**

- [ ] 循迹 PID 生效（kp=1.2, kd=0.1 起步）
- [ ] 小车能稳定沿黑线跑
- [ ] 不过度左右振荡
- [ ] 弯道不冲出线
- [ ] 速度可逐步提高（200→300 mm/s）

## 9. OLED 状态显示

- [ ] 第 1 页：模式/电池/偏差/速度
- [ ] 第 2 页：PID 参数
- [ ] 第 3 页：传感器状态
- [ ] 100~200ms 刷新，不闪烁

## 10. 工程化检查

- [ ] 状态机正常工作（BOOT→IDLE→TEST→FOLLOW→ERROR）
- [ ] UART 命令可切换模式（`mode follow_pid` 等）
- [ ] UART 可查询状态（`status`）
- [ ] UART 可修改 PID 参数（`pid speed_l 0.5 0.4 0.0`）
- [ ] 按键可启动/停止/切换模式
- [ ] 低电压时 OLED 显示警告并限制 PWM
