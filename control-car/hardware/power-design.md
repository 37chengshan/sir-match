# 电源设计：地猛星 + TB6612 小车供电方案

## 1. 电源架构

```
7.4V 锂电池
├─ TB6612 VM       → 电机动力电（不经稳压）
├─ 电源模块 5V     → 灰度传感器（如需5V）
├─ 电源模块 3.3V   → 地猛星开发板 + 编码器 + OLED
└─ 电池电压采样    → ADC 监测（分压后进 MSPM0）
```

## 2. 关键原则

1. **逻辑电与动力电分开**
   - 地猛星、编码器、灰度、OLED 从电源模块 3.3V/5V 取电
   - TB6612 VM 从电池直接取电（不经电源模块）
   - MSPM0 不要直接给电机供电

2. **共地**
   - 所有模块的 GND 必须连在一起
   - 电池 GND → 电源模块 GND → TB6612 GND → MSPM0 GND

3. **滤波**
   - TB6612 VM 引脚附近并联大电解电容（100~470μF / 16V）
   - 电源模块输出端加 0.1μF 瓷片电容

## 3. TB6612 供电

| TB6612 引脚 | 供电源 | 电压 | 说明 |
|------------|--------|------|------|
| VM | 电池正极 | 6.6~8.4V | 电机动力电，不经稳压 |
| VCC | 电源模块 3.3V | 3.3V | 逻辑电 |
| GND | 共地 | 0V | 与电池地、逻辑地相连 |

VM 电流需求：堵转时单电机可达 1~2A，双电机 2~4A。

## 4. 电池选择

| 参数 | 建议值 |
|------|--------|
| 类型 | 2S 锂电池（7.4V 标称） |
| 容量 | ≥ 2000mAh |
| 放电倍率 | ≥ 5C |
| 充电器 | 配 12.6V 专用充电器 |

建议买 2 套轮换使用。

## 5. 电池电压监测

用电阻分压后进 MSPM0 ADC：

```
电池正极 ── R1(20k) ──┬── R2(10k) ── GND
                       │
                    ADC 输入

V_adc = V_battery * R2 / (R1 + R2) = V_battery * 10k / 30k = V_battery / 3

V_battery = V_adc * 3
```

例：电池 7.4V → ADC 约 2.47V（在 MSPM0 3.3V ADC 量程内）。

## 6. 上电检查

| 检查项 | 方法 | 预期 |
|--------|------|------|
| 电池电压 | 万用表测电池正负极 | 6.6~8.4V |
| VM 电压 | 万用表测 TB6612 VM-GND | 同电池电压 |
| 电源模块 5V | 万用表 | 4.8~5.2V |
| 电源模块 3.3V | 万用表 | 3.2~3.4V |
| 不短路 | 万用表蜂鸣档 | 各电源轨对地不导通 |

## 7. 故障保护

- 电池低电压告警阈值：6.6V（2S 锂电池单节 3.3V 时建议充电）
- 代码中定期读 ADC，低于阈值时 OLED 显示警告并限制 PWM 输出
- 电机堵转时电流大，代码应设 PWM 最大占空比限制
