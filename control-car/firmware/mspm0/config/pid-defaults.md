# PID 默认参数与调参记录

> 这些参数不是最终值，只是安全起步值。真实参数必须根据车重、轮径、电机减速比、循迹传感器高度、地面材质重新整定。

## 1. 控制结构

推荐两层控制：

```text
循迹偏差 line_error
→ 转向 PID
→ 左右轮目标速度差

左右轮目标速度
→ 左右轮速度 PID
→ 左右轮 PWM 输出
```

## 2. 起步参数

| 控制器 | kp | ki | kd | 说明 |
|---|---:|---:|---:|---|
| 左轮速度 PID | 0.8 | 0.02 | 0.00 | 先只用 P，再逐步加 I |
| 右轮速度 PID | 0.8 | 0.02 | 0.00 | 左右初值一致 |
| 循迹转向 PID | 1.2 | 0.00 | 0.10 | 先 P+D，通常不急着加 I |

## 3. 限幅建议

| 参数 | 建议初值 |
|---|---:|
| 最大 PWM | 300/1000 |
| 起步基础速度 | 120 mm/s |
| 稳定后基础速度 | 180~250 mm/s |
| 转向修正最大值 | 基础速度的 60% |
| 积分限幅 | 输出上限的 20% |

## 4. 调参顺序

1. 不开循迹，只测左右轮速度；
2. 先调速度 P，让速度能跟上但不明显振荡；
3. 再加少量 I，减少稳态误差；
4. 暂时不加速度 D；
5. 再低速跑开环循迹；
6. 调循迹 Kp，让小车能回到线；
7. 增加 Kd，减少左右摇摆；
8. 最后逐步提高基础速度。

## 5. 常见现象

| 现象 | 可能原因 | 调整 |
|---|---|---|
| 左右摆动很大 | 循迹 Kp 太大或速度太快 | 降 Kp 或降基础速度 |
| 过弯冲出线 | 基础速度太高，Kd 不够 | 降速，适当加 Kd |
| 直线偏航 | 左右轮速度不一致 | 先调速度环和方向系数 |
| 反应很慢 | Kp 太小 | 增大 Kp |
| 长时间偏一边 | 机械偏差或稳态误差 | 检查轮子，再少量加 I |
