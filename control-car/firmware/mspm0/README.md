# MSPM0G3507 固件工作区

本目录用于放置 `control-car` 的 MSPM0G3507 固件工程、接口骨架、配置说明和测试记录。

## 当前原则

1. 先文档化 pin map、串口协议、PID 默认参数，再写真实工程。
2. 不确认实际 PCB 引脚时，不要硬编码引脚。
3. 不要把所有逻辑写进 `main.c`。
4. 固件建议按以下模块拆分：

```text
app/        状态机、任务调度、模式切换
board/      引脚、时钟、SysConfig 适配
drivers/    电机、编码器、循迹、OLED、按键、蜂鸣器、ADC
control/    PID、速度环、循迹环
protocol/   UART 调参协议
utils/      环形缓冲、滤波、限幅工具
config/     pin map、PID 默认值、串口协议文档
tests/      电机、编码器、循迹、PID 调试记录
```

## 第一版固件验收

- MSPM0G3507 工程能下载运行；
- 串口能输出 BOOT 日志；
- OLED 能显示当前模式；
- 左右电机能独立正反转和调速；
- 编码器能读到方向和速度；
- 灰度传感器能输出黑白状态和偏差；
- 可以通过串口调整速度和 PID 参数。
