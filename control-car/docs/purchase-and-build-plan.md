# 采购与搭建计划：MSPM0G3507 循迹小车 + K230 视觉

> 结论：第一批不要买太多复杂模块。先把 MSPM0G3507 小车的底层闭环能力做稳：主控、电源、电机、编码器、循迹、PID、串口/OLED 调参。视觉主线统一为 **庐山派 K230 / CanMV K230**，但 K230 放在底层闭环稳定之后接入。

---

## 1. 采购策略

采购按四批推进：

```text
第一批：能跑起来
MSPM0G3507 + 小车 PCB + 编码器电机 + TB6612 + 灰度循迹 + 电源 + 串口/OLED

第二批：能稳定调参
逻辑分析仪 + 备用模块 + 蓝牙/IMU/超声波 + 场地材料

第三批：K230 视觉接入
庐山派 K230 + TF 卡 + 读卡器 + 摄像头支架 + 独立 5V 稳压 + UART 线

第四批：冲高分扩展
云台 + K230 靶面识别/二维码/AprilTag/YOLO + 经老师确认的安全光源方案
```

核心原则：

- 先闭环，后扩展；
- 先两轮差速，后复杂底盘；
- 先串口/OLED 调参，后无线通信；
- 先普通灰度循迹，后 K230 视觉；
- 先 K230 基础视觉，后 YOLO/kmodel；
- 先稳定，后高分。

---

## 2. 第一批必买：小车闭环控制基础件

### 2.1 主控与调试

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 立创·地猛星 MSPM0G3507 开发板 | 1~2 块 | 必买 | 主控平台，接小车 PCB 和模块 |
| TI 官方 LP-MSPM0G3507 LaunchPad | 1 块 | 推荐 | 学 TI 官方 SDK、SysConfig、例程 |
| Type-C 数据线 | 2 根 | 必买 | 下载、供电、串口 |
| USB 转串口模块 CH340/CP2102 | 1~2 个 | 必买 | 调参、打印日志 |
| 排针/排母 | 若干 | 必买 | 焊接开发板和扩展接口 |
| 杜邦线：公对公/公对母/母对母 | 各 1 包 | 必买 | 模块连接 |

注意：MSPM0G3507 不使用 ST-LINK 下载。ST-LINK 留给 STM32 使用。

### 2.2 小车机械与驱动

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 老师提供的循迹小车 PCB | 1~2 块 | 必买 | 小车主底板/扩展底座 |
| N20 带编码器减速电机 | 2 个 | 必买 | 左右轮驱动和测速 |
| N20 电机支架 | 2 个 | 必买 | 固定电机 |
| N20 车轮 | 2 个 | 必买 | 与电机轴匹配 |
| 万向轮 | 1 个 | 必买 | 两轮差速结构支撑 |
| TB6612FNG 电机驱动模块 | 2 个 | 必买 | 一个使用，一个备用 |
| 铜柱、螺丝、螺母、扎带 | 若干 | 必买 | 固定 PCB、模块、电池、线束 |
| 小车底板/亚克力板/3D 打印底板 | 1 套 | 视 PCB 而定 | PCB 不能直接承重时使用 |

推荐底盘：

```text
左 N20 编码器电机 + 右 N20 编码器电机 + 后万向轮
```

不建议第一版直接做四驱，因为调参和机械误差更复杂。

### 2.3 循迹传感器

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 8 路灰度循迹传感器阵列 | 1 个 | 推荐必买 | 做加权偏差和转向 PID |
| 5 路红外/灰度循迹模块 | 1 个 | 可选备用 | 入门测试或备件 |
| 黑色电工胶带，约 18mm | 2 卷 | 必买 | 制作循迹线和靶面边框 |
| 白色哑光喷绘布/白色场地布 | 1 张 | 推荐 | 模拟正式比赛场地 |

优先买 8 路灰度阵列，因为后续可以计算偏差值：

```text
-7 -5 -3 -1 +1 +3 +5 +7
```

比单纯 3 路/5 路判断更适合 PID。

### 2.4 电源系统

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 18650 电池 | 4~6 节 | 必买 | 小车主电源 |
| 2 节 18650 电池盒 | 1~2 个 | 必买 | 约 7.4V 输入 |
| 18650 充电器 | 1 个 | 必买 | 安全充电 |
| 5V 降压模块，3A 以上 | 2 个 | 必买 | 给传感器、OLED、K230、舵机扩展供电 |
| 3.3V 降压/稳压模块 | 1~2 个 | 推荐 | 给 3.3V 外设备用 |
| 船型开关/拨动开关 | 4~6 个 | 必买 | 电机、小车控制、扩展模块独立开关 |
| LED 电源指示灯模块 | 4~6 个 | 必买 | 显示供电状态 |
| 470uF/1000uF 电解电容 | 若干 | 必买 | 抑制电机启动导致复位 |
| JST/XH 插头线 | 若干 | 推荐 | 电池和模块连接更牢固 |

建议电源结构：

```text
电池 7.4V
├── TB6612 VM → 电机
├── 5V/3A 降压 → K230 / 舵机 / 扩展模块
└── 3.3V/开发板稳压 → MSPM0 逻辑
```

关键要求：

- 电机电源与逻辑电源分清；
- K230 建议独立 5V 稳压支路；
- 所有模块必须共地；
- 电机驱动附近加大电容；
- 电池和线束必须固定，避免跑车时松动。

## 3. 第二批推荐：调试和稳定性提升

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| USB 逻辑分析仪 8 通道 | 1 个 | 强烈推荐 | 看 UART/I2C/PWM/编码器波形 |
| 万用表 | 1 个 | 必买 | 测电压、电源、通断 |
| 蓝牙串口模块 HC-05/HM-10 | 1 个 | 推荐 | 手机调参，后期再上 |
| IMU 模块 | 1 个 | 推荐 | 姿态/偏航角扩展 |
| 超声波模块 | 1 个 | 推荐 | 避障训练 |
| OLED 0.96 寸 I2C | 1~2 个 | 必买/已有可复用 | 状态显示 |
| 按键模块 | 4~6 个 | 必买 | 模式切换、启动停止 |
| 有源蜂鸣器 | 1~2 个 | 可复用 | 状态提示 |
| 热熔胶枪 | 1 个 | 推荐 | 固定线束和模块 |
| 洞洞板/万能板 | 2~3 块 | 推荐 | 临时转接电路 |

逻辑分析仪非常重要，优先级高于蓝牙。它能直接帮助定位：

- 编码器有没有脉冲；
- I2C OLED 有没有 ACK；
- MSPM0 ↔ K230 UART 有没有数据；
- PWM 频率和占空比是否正确。

## 4. 第三批：K230 视觉接入

这部分等小车底层稳定后再接入，但 TF 卡和读卡器可以先买，因为没有 TF 卡基本不能完整练 K230。

### 4.1 K230 必买件

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 立创·庐山派 K230 / CanMV K230 | 1 个 | 已定主线 | 视觉协处理器 |
| microSD / TF 卡 | 1~2 张 | 必买 | K230 启动系统、保存脚本/模型 |
| TF 读卡器 | 1 个 | 必买 | 烧录 K230 镜像 |
| USB-C 数据线 | 1~2 根 | 必买 | K230 供电/调试 |
| 摄像头支架 | 1 个 | 必买 | 固定视角 |
| 5V 降压模块，2A 以上 | 1 个 | 必买 | 给 K230 独立供电 |
| GH1.25 转杜邦线 / 杜邦线 | 若干 | 推荐 | K230 UART 接 MSPM0 |
| 补光 LED | 1 个 | 推荐 | 提高识别稳定性 |

### 4.2 TF 卡规格

| 项目 | 建议 |
|---|---|
| 容量 | 8GB 或 16GB，32GB 也可以 |
| 速度 | Class 10 / U1 及以上 |
| 最低写入 | 约 10MB/s |
| 文件用途 | K230 系统镜像、CanMV 脚本、模型文件 |

不要买杂牌、无速度标识、老旧 2GB 卡。烧录镜像会清空 TF 卡内容。

### 4.3 K230 接入优先级

```text
K230 单板启动
→ 摄像头预览
→ 色块/二维码/AprilTag/黑线识别
→ UART 输出 VISION 帧
→ MSPM0 解析并 OLED 显示
→ 视觉辅助循迹/云台
```

- 不要照向人眼、皮肤、镜面、金属反光面。

## 6. 暂时不要买/不要优先买

| 器材 | 原因 |
|---|---|
| 大量不同开发板 | 会分散精力，先玩透 MSPM0G3507。 |
| 树莓派 5 + 摄像头 | 功能强但电源、系统、封箱、调试复杂，不作为当前主线。 |
| OV2640 裸摄像头模块 | 需要 DCMI/并口采集，和 STM32F103C8T6/当前 MSPM0 主线不匹配。 |
| 多套视觉模块同时并行 | 文档和代码会混乱；当前统一 K230。 |
| L298N | 体积大、损耗高，不如 TB6612。 |
| 无编码器电机 | 不适合 PID 和电赛控制训练。 |

## 7. 采购一致性结论

当前小车项目统一口径：

```text
主控：地猛星 MSPM0G3507
下载：XDS110
电机：N20 编码器电机
驱动：TB6612FNG
基础循迹：5路/8路灰度
视觉主线：庐山派 K230 / CanMV K230
K230 必需：TF 卡 + 读卡器 + 独立 5V 稳压 + 摄像头支架
通信：UART 115200 8N1
OpenMV：不再作为主线
```
