# 采购与搭建计划：MSPM0G3507 循迹小车

> 结论：第一批不要买太多复杂模块。先把 MSPM0G3507 小车的底层闭环能力做稳：主控、电源、电机、编码器、循迹、PID、串口/OLED 调参。

---

## 1. 采购策略

采购按三批推进：

```text
第一批：能跑起来
MSPM0G3507 + 小车 PCB + 编码器电机 + TB6612 + 灰度循迹 + 电源 + 串口/OLED

第二批：能稳定调参
逻辑分析仪 + 备用模块 + 蓝牙/IMU/超声波 + 场地材料

第三批：冲高分扩展
云台 + OpenMV/视觉模块 + 靶面材料 + 经老师确认的安全光源方案
```

核心原则：

- 先闭环，后扩展；
- 先两轮差速，后复杂底盘；
- 先串口/OLED 调参，后无线通信；
- 先普通循迹，后视觉/云台；
- 先稳定，后高分。

---

## 2. 第一批必买：小车闭环控制基础件

### 2.1 主控与调试

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 立创·天猛星 MSPM0G3507 开发板 | 1~2 块 | 必买 | 主控平台，接小车 PCB 和模块 |
| TI 官方 LP-MSPM0G3507 LaunchPad | 1 块 | 推荐 | 学 TI 官方 SDK、SysConfig、例程 |
| Type-C 数据线 | 2 根 | 必买 | 下载、供电、串口 |
| USB 转串口模块 CH340/CP2102 | 1~2 个 | 必买 | 调参、打印日志 |
| 排针/排母 | 若干 | 必买 | 焊接开发板和扩展接口 |
| 杜邦线：公对公/公对母/母对母 | 各 1 包 | 必买 | 模块连接 |

注意：MSPM0G3507 不使用 ST-LINK 下载。ST-LINK 留给 STM32 使用。

### 2.2 小车机械与驱动

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 老师提供的循迹小车 PCB | 1~2 块 | 必买 | 小车主底板/扩展底座 |
| N20 带编码器减速电机 | 2 个 | 必买 | 左右轮驱动和测速 |
| N20 电机支架 | 2 个 | 必买 | 固定电机 |
| N20 车轮 | 2 个 | 必买 | 与电机轴匹配 |
| 万向轮 | 1 个 | 必买 | 两轮差速结构支撑 |
| TB6612FNG 电机驱动模块 | 2 个 | 必买 | 一个使用，一个备用 |
| 铜柱、螺丝、螺母、扎带 | 若干 | 必买 | 固定 PCB、模块、电池、线束 |
| 小车底板/亚克力板/3D 打印底板 | 1 套 | 视 PCB 而定 | PCB 不能直接承重时使用 |

推荐底盘：

```text
左 N20 编码器电机 + 右 N20 编码器电机 + 后万向轮
```

不建议第一版直接做四驱，因为调参和机械误差更复杂。

### 2.3 循迹传感器

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 8 路灰度循迹传感器阵列 | 1 个 | 推荐必买 | 做加权偏差和转向 PID |
| 5 路红外/灰度循迹模块 | 1 个 | 可选备用 | 入门测试或备件 |
| 黑色电工胶带，约 18mm | 2 卷 | 必买 | 制作循迹线和靶面边框 |
| 白色哑光喷绘布/白色场地布 | 1 张 | 推荐 | 模拟正式比赛场地 |

优先买 8 路灰度阵列，因为后续可以计算偏差值：

```text
-7 -5 -3 -1 +1 +3 +5 +7
```

比单纯 3 路/5 路判断更适合 PID。

### 2.4 电源系统

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 18650 电池 | 4~6 节 | 必买 | 小车主电源 |
| 2 节 18650 电池盒 | 1~2 个 | 必买 | 约 7.4V 输入 |
| 18650 充电器 | 1 个 | 必买 | 安全充电 |
| 5V 降压模块，3A 以上 | 2 个 | 必买 | 给传感器、舵机、部分模块供电 |
| 3.3V 降压/稳压模块 | 1~2 个 | 推荐 | 给 3.3V 外设备用 |
| 船型开关/拨动开关 | 4~6 个 | 必买 | 电机、小车控制、扩展模块独立开关 |
| LED 电源指示灯模块 | 4~6 个 | 必买 | 显示供电状态 |
| 470uF/1000uF 电解电容 | 若干 | 必买 | 抑制电机启动导致复位 |
| JST/XH 插头线 | 若干 | 推荐 | 电池和模块连接更牢固 |

建议电源结构：

```text
电池 7.4V
├── TB6612 VM → 电机
├── 5V 降压 → 传感器/OLED/舵机扩展
└── 3.3V/开发板稳压 → MSPM0 逻辑
```

关键要求：

- 电机电源与逻辑电源分清；
- 所有模块必须共地；
- 电机驱动附近加大电容；
- 电池和线束必须固定，避免跑车时松动。

## 3. 第二批推荐：调试和稳定性提升

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| USB 逻辑分析仪 8 通道 | 1 个 | 强烈推荐 | 看 UART/I2C/PWM/编码器波形 |
| 万用表 | 1 个 | 必买 | 测电压、电源、通断 |
| 蓝牙串口模块 HC-05/HM-10 | 1 个 | 推荐 | 手机调参，后期再上 |
| IMU 模块 | 1 个 | 推荐 | 姿态/偏航角扩展 |
| 超声波模块 | 1 个 | 推荐 | 避障训练 |
| OLED 0.96 寸 I2C | 1~2 个 | 必买/已有可复用 | 状态显示 |
| 按键模块 | 4~6 个 | 必买 | 模式切换、启动停止 |
| 有源蜂鸣器 | 1~2 个 | 可复用 | 状态提示 |
| 热熔胶枪 | 1 个 | 推荐 | 固定线束和模块 |
| 洞洞板/万能板 | 2~3 块 | 推荐 | 临时转接电路 |

逻辑分析仪非常重要，优先级高于蓝牙。它能直接帮助定位：

- 编码器有没有脉冲；
- I2C OLED 有没有 ACK；
- UART 命令有没有发出；
- PWM 频率和占空比是否正确。

## 4. 第三批扩展：云台、视觉、瞄准

这部分等小车底层稳定后再买。

### 4.1 云台

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 二自由度云台支架 | 1 套 | 推荐 | 水平 + 俯仰控制 |
| MG90S 金属舵机 | 2~4 个 | 推荐 | 比 SG90 更稳 |
| PCA9685 舵机驱动板 | 1 个 | 推荐 | I2C 多路 PWM 控制舵机 |
| 舵机专用 5V 降压模块 | 1 个 | 必需 | 舵机不能直接吃开发板 5V |
| 云台固定板/3D 打印件 | 1 套 | 推荐 | 固定摄像头和光源 |

第一版建议：

```text
MSPM0G3507 → I2C → PCA9685 → 双 MG90S 云台
```

### 4.2 视觉

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| OpenMV H7/H7 Plus | 1 个 | 推荐 | 快速做矩形识别/色块识别 |
| K210/K230 视觉模块 | 1 个 | 进阶 | 性能更强，但学习成本更高 |
| 树莓派 5 + 摄像头 | 1 套 | 高阶 | 强，但电源和软件复杂 |
| 摄像头支架 | 1 个 | 必需 | 固定视角 |
| 补光 LED | 1 个 | 推荐 | 提高识别稳定性 |

建议先 OpenMV，不建议第一版直接树莓派。

### 4.3 靶面与光源

| 器材 | 建议数量 | 优先级 | 用途 |
|---|---:|---|---|
| 紫外感光纸 A4 | 若干 | 后期 | 模拟 E 题靶面 |
| 红色油性记号笔 | 1 支 | 后期 | 画靶心和圆环 |
| 靶面支架/夹子/硬纸板 | 1 套 | 后期 | 固定靶纸 |
| 激光防护眼镜 | 1 副以上 | 必需 | 光源相关实验保护 |
| 经老师确认的低功率合规光源 | 1 个 | 后期 | 用于正式靶面测试 |

安全原则：

- 早期用普通 LED 或低风险可见光指示模块替代；
- 任何激光/蓝紫光相关测试必须经过老师或实验室安全确认；
- 不要照向人眼、皮肤、镜面、金属反光面。

## 5. 暂时不要买/不要优先买

| 器材 | 原因 |
|---|---|
| 大量不同开发板 | 会分散精力，先玩透 MSPM0G3507 |
| L298N | 发热大、效率低，不如 TB6612FNG |
| 无编码器电机 | 后续 PID 训练价值低 |
| 四驱底盘 | 第一版机械和调参复杂 |
| 麦克纳姆轮 | 不适合多数循迹规则和当前训练目标 |
| 树莓派 5 | 前期系统复杂度过高 |
| K230/K210 | 视觉算法未开始时不急 |
| 高功率光源/激光模块 | 有安全风险，且早期没必要 |

## 7. 验收路线

| 阶段 | 验收目标 |
|---|---|
| 1 | MSPM0 工程可下载、串口有日志、OLED 有显示 |
| 2 | 左右电机可独立正反转、调速、停止 |
| 3 | 编码器方向正确，速度数值稳定 |
| 4 | 循迹传感器能稳定判断黑白线 |
| 5 | 小车低速 open-loop 跑完整圈 |
| 6 | 速度 PID 后左右轮速度一致性提升 |
| 7 | 转向 PID 后循迹更稳、振荡减少 |
| 8 | 参数可通过串口/OLED 快速查看与调整 |

## 8. 常见故障排查

| 现象 | 优先检查 |
|---|---|
| 电机一转开发板复位 | 电源降压能力、电机电容、共地、线太细 |
| 电机只响不转 | PWM 太小、电机死区、TB6612 STBY、供电不足 |
| 左右轮方向相反 | 电机线反接或方向配置错误 |
| 编码器没有数据 | 编码器供电、A/B 相接线、中断/定时器配置 |
| OLED 不显示 | I2C 地址、SCL/SDA 接线、上拉、电源电压 |
| 循迹误判严重 | 传感器高度、环境光、黑线材质、阈值未标定 |
| 小车左右摇摆 | 转向 Kp 太大、速度太快、传感器太靠前或太靠后 |
| 弯道冲出线 | 基础速度太高、急弯未降速、Kd 不足 |
| 串口乱码 | 波特率、GND、TX/RX 交叉、串口电平 |

---

## 9. 购买后的第一天任务

1. 焊接排针排母；
2. 只接 MSPM0，不接电机，先完成点灯、串口、OLED；
3. 单独测试 TB6612 和一个电机；
4. 测量 5V 降压模块输出；
5. 确认所有模块共地；
6. 小车上电前用万用表测短路；
7. 第一次跑车时架空车轮，不要直接放地上；
8. 串口打印电机 PWM、编码器、传感器状态。

---
