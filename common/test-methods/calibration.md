# 标定流程

## 1. 作用

标定用于把 ADC/DAC/信号源的系统误差降到可接受范围。G题中至少需要 ADC 零点、幅度和相位标定。

## 2. ADC 零点标定

步骤：

1. 输入端短接到 1.65V 偏置；
2. 连续采样 N 次；
3. 计算平均值；
4. 保存为 `adc_offset`；
5. 后续算法先减 offset。

## 3. 幅度标定

步骤：

1. AD9833 输出 1kHz 正弦；
2. 示波器测真实 Vpp；
3. ADC 计算 Vpp；
4. 计算比例系数；
5. 对 5kHz、10kHz、50kHz 重复测试。

## 4. 相位标定

步骤：

1. Vin 和 Vout 接同一个信号；
2. 对两个通道做 Goertzel；
3. 理论相位差应为 0°；
4. 实测相位差作为通道相位偏差；
5. 后续测量时扣除。

## 5. DAC 标定

步骤：

1. 输出 0、1/4、1/2、3/4、满量程附近电平；
2. 万用表或示波器测电压；
3. 建立 DAC code 到电压的对应关系；
4. 检查是否削顶。

## 6. 必须记录

```text
日期
硬件版本
固件 commit
测试仪器
频率
示波器 Vpp
ADC Vpp
相位偏差
标定系数
备注
```

## 7. 常见坑

- 偏置电压不稳；
- 用 ADC 数据标定 ADC 自己，没有示波器对照；
- 相位标定时 Vin/Vout 接线长度差太大；
- 不同频率下前端增益不同，但只标定一个频点。
