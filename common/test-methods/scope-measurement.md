# 示波器测量方法

## 1. 作用

示波器是判断硬件是否真实工作的第一工具。G题中不要只相信 ADC 数据，必须先用示波器确认 Vin、Vout、DAC 输出和 AD9833 输出。

## 2. G题必测点

| 测试点 | 作用 |
|---|---|
| AD9833_OUT | 检查信号源频率和原始幅度 |
| BUF_OUT | 检查缓冲后是否失真 |
| VIN_ADC_IN | 检查进入 ADC 前是否在 0~3.3V |
| VOUT_ADC_IN | 检查输出采样前是否在安全范围 |
| DAC_OUT | 检查复现波形 |
| 1.65V_REF | 检查偏置稳定性 |
| 3.3V / 5V | 检查电源纹波和跌落 |

## 3. 测 Vpp

1. 探头接测试点，地夹接系统 GND；
2. 通道选择 DC 耦合；
3. 调整垂直档位；
4. 用 Measure 功能读 Vpp；
5. 同时记录频率。

## 4. 测相位

1. CH1 接 Vin；
2. CH2 接 Vout；
3. 两个通道使用相同探头倍率；
4. 使用示波器相位测量或游标测时间差；
5. 记录相位差，与 Goertzel 计算值对比。

## 5. 常见坑

- 探头倍率设置不一致；
- AC/DC 耦合选错；
- 地夹导致短路；
- 面包板接触不良；
- 示波器带宽限制影响高频波形；
- 探头地线太长，引入高频噪声。

## 6. 保存要求

每个关键阶段保存：

- 示波器截图；
- 频率；
- Vpp；
- 相位差；
- 硬件版本；
- 固件 commit。
