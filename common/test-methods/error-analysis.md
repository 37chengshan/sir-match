# 误差分析方法

## 1. 作用

误差分析是电赛报告得分的重要部分。不要只写“存在误差”，要说明误差来源、影响方向和改进措施。

## 2. G题主要误差来源

| 来源 | 影响 |
|---|---|
| 电阻电容误差 | 已知模型理论值和实测值不一致 |
| 电感误差 | RLC 中心频率偏移 |
| AD9833 输出幅度变化 | 扫频时 Vin 不恒定 |
| 运放带宽和压摆率 | 高频幅值下降、相位滞后 |
| ADC 量化误差 | 幅值和相位计算误差 |
| ADC 通道不同步 | 相位误差 |
| 1.65V 偏置波动 | 直流偏置误差 |
| 面包板寄生参数 | 高频段异常 |
| 示波器探头误差 | 对照测量误差 |

## 3. 小车主要误差来源

| 来源 | 影响 |
|---|---|
| 左右电机差异 | 跑偏 |
| 轮胎打滑 | 速度估计不准 |
| 编码器抖动 | 速度噪声 |
| 灰度传感器高度变化 | 循迹误判 |
| 电池电压下降 | PWM 同值速度变慢 |
| 地面材质变化 | 反射率变化 |

## 4. 报告写法模板

```text
理论值：
测量值：
绝对误差：
相对误差：
主要误差来源：
改进措施：
```

## 5. 推荐图表

- 理论幅频曲线 vs 实测幅频曲线；
- 理论相频曲线 vs 实测相频曲线；
- PID 收敛曲线；
- 不同频率下 Vin 稳定性表；
- RLC 参数理论值/测量值/误差表。

## 6. 注意

报告中不要过度承诺精度。对面包板原型，应主动说明高频寄生、运放带宽和采样相位会影响高频段误差。
