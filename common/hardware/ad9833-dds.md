# AD9833 DDS 信号源

## 1. 作用

AD9833 用作可编程信号源，适合 G题输出 1kHz~50kHz 正弦扫频信号。

## 2. 基本能力

- SPI 控制；
- 输出正弦波、三角波、方波；
- 频率分辨率高；
- 模块常见供电为 3.3V 或 5V，逻辑电平需确认。

## 3. 推荐接法

```text
MSPM0G3507 SPI_SCLK  → AD9833 SCLK
MSPM0G3507 SPI_MOSI  → AD9833 SDATA
MSPM0G3507 GPIO_CS   → AD9833 FSYNC
3.3V / 5V            → VCC，按模块说明
GND                  → GND
```

如果模块 VCC 用 5V，但 SPI 逻辑不兼容 3.3V，必须加电平转换或换 3.3V 兼容模块。

## 4. 输出链路

AD9833 输出不要直接接被测电路。推荐：

```text
AD9833 OUT → RC低通/重构 → 运放缓冲 → 数字电位器/可变增益 → 被测电路
```

## 5. G题要求

- 输出覆盖 1kHz~50kHz；
- 输出尽量稳定；
- 最终被测输入点 Vin 应为 2Vpp 左右；
- 频率切换后需要等待稳定再采样；
- 扫频时串口记录当前频率和实测 Vin。

## 6. 最小测试

1. 固定输出 1kHz；
2. 示波器确认波形；
3. 切换 5kHz、10kHz、50kHz；
4. 测输出幅度是否随频率明显变化；
5. 接入运放缓冲后再次测量。

## 7. 常见坑

- FSYNC 极性或时序错误；
- SPI 模式配置错误；
- 输出幅度太小，以为代码没工作；
- 模块自带滤波不足，高频镜像明显；
- 负载太重导致输出畸变；
- 输出没有偏置/缓冲，后级 ADC 采样异常。

## 8. 替代方案

- 低频查表 DAC 输出；
- 外置信号发生器训练；
- 更高性能 DDS 模块。

比赛方案中仍优先 AD9833，因为它便宜、易控、资料多、适合扫频。
