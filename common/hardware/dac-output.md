# DAC 查表输出

## 1. 作用

DAC 输出用于产生复现波形。G题中，MSPM0G3507 通过 DAC 查表输出学习到的未知电路响应。

典型链路：

```text
算法生成波形表 → DAC定时输出 → RC重构滤波 → 运放缓冲 → 复现输出端
```

## 2. 输出类型

优先顺序：

1. 正弦波；
2. 多谐波合成波；
3. 矩形波近似；
4. 任意周期波；
5. m序列卷积复现结果。

## 3. 推荐链路

```text
MSPM0G3507 DAC0 → RC低通 → 运放电压跟随/放大 → 输出端
```

如果要输出交流信号，需要处理：

- DAC 中点偏置；
- 输出幅度缩放；
- 运放带宽；
- 后级负载。

## 4. 查表策略

第一版建议：

- 256 点正弦表；
- 定时器固定频率触发 DAC；
- 输出频率由 `sample_rate / table_len` 决定；
- 幅度和偏置通过表值计算。

## 5. G题波形复现策略

正弦波：

```text
目标频率 f
学习得到 H(jw)
计算幅值和相位
生成带相位偏移的正弦表
DAC 输出
```

矩形波：

```text
分解前 5~10 个谐波
每个谐波乘以 H(jw)
叠加后归一化
DAC 输出
```

## 6. 常见坑

- DAC 输出没有缓冲，负载一接就变形；
- 表值超出 0~4095；
- 合成波没有归一化，削顶严重；
- RC 重构滤波截止频率过低，高频谐波被削掉；
- DAC 更新率不足，矩形波边沿不可能理想；
- 示波器 AC/DC 耦合设置错误导致误判。

## 7. 最小测试

1. 输出固定中点，测电压是否稳定；
2. 输出 1kHz 正弦表；
3. 改变幅度，确认不削顶；
4. 接运放缓冲后再次测量；
5. 用示波器截图保存结果。

## 8. 验收标准

- 1kHz 正弦无明显削顶；
- 输出频率和表计算一致；
- 接入后级负载后幅值变化不大；
- 复现波形趋势与 MATLAB 仿真一致。
