# 数字电位器

## 1. 作用

数字电位器用于通过单片机控制模拟幅度。G题中可用于 AD9833 输出后的自动幅度调节，使被测输入或输出稳定到目标 Vpp。

## 2. 推荐型号

| 型号 | 接口 | 说明 |
|---|---|---|
| MCP41010 | SPI | 10kΩ，常见，适合练习 |
| AD5160 | I2C/SPI 版本按模块而定 | 可替代 |
| X9C103 | 三线控制 | 简单但速度较慢 |

## 3. 典型链路

```text
AD9833 OUT → 运放缓冲 → 数字电位器衰减/调幅 → 运放缓冲 → 被测电路
```

或：

```text
数字电位器设置运放反馈电阻 → 可变增益放大器
```

## 4. 接线注意

- 数字端逻辑电平必须兼容 MSPM0G3507 的 3.3V；
- 模拟端电压不能超过数字电位器允许范围；
- SPI 总线可与 AD9833 共用，但 CS 必须独立；
- 模拟信号要先确认是否在 0~VDD 范围内。

## 5. 控制策略

第一版建议不用复杂 PID，先分段调节：

```text
误差 > 20%：大步调节
误差 5%~20%：小步调节
误差 < 5%：保持
```

跑稳后再切 PID。

## 6. 常见坑

- 把交流负半周直接送进单电源数字电位器；
- 数字电位器阻值太大，和后级输入阻抗形成额外分压；
- SPI CS 没分开，AD9833 和数字电位器同时响应；
- 步进分辨率不足，导致幅度控制抖动；
- 输出阻抗太高，后级负载一接就变。

## 7. 最小测试

1. 串口输入目标档位；
2. 数字电位器阻值或输出幅度随档位变化；
3. AD9833 固定 1kHz 输出；
4. 改变档位，示波器能看到幅值变化；
5. 闭环控制能把 Vpp 拉向目标值。

## 8. G题验收

- 输出稳定到 2Vpp 附近；
- 频率从 1kHz 切到 50kHz 后仍能重新调节；
- 串口输出当前档位、Vpp、误差；
- 不出现持续振荡。
