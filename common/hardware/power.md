# 电源模块

## 1. 作用

电源是电赛系统稳定性的底座。G题涉及模拟采样，小车涉及电机干扰，两者都对供电和地线非常敏感。

## 2. 基础电源架构

推荐：

```text
输入电源
  ├── DC-DC 5V：显示、部分模块、运放供电
  ├── LDO 3.3V：MSPM0G3507、AD9833逻辑、OLED
  ├── 模拟3.3V/参考源：ADC前端、基准、偏置
  └── 电机电源：小车电机，单独走线
```

## 3. 必备模块

| 模块 | 用途 |
|---|---|
| LM2596 / MP1584 | 降压到 5V |
| AMS1117 / TLV75533 | 3.3V LDO |
| TL431 / REF3025 | 参考电压 |
| 0.1µF + 10µF 去耦 | 每个模块附近供电稳定 |
| 自恢复保险 / 限流 | 防短路 |
| 电源开关 | 模块独立断电 |

## 4. G题电源重点

- 模拟前端尽量用干净电源；
- 1.65V 偏置要稳定；
- AD9833、运放、ADC 前端附近加去耦；
- 模拟地走线短，不要让大电流经过采样地；
- 示波器地夹连接时注意共地。

## 5. 小车电源重点

- 电机电源和主控电源分开；
- TB6612 / 电机驱动附近加大电容；
- 电池输入加开关和保险；
- 电机线远离 ADC、I2C、OLED 线；
- 主控复位多半是电源跌落或地弹噪声。

## 6. 最小测试

1. 空载测 5V 和 3.3V；
2. 插上主控后测电压；
3. AD9833 输出时测电压纹波；
4. 小车电机启动瞬间测主控是否复位；
5. 模拟采样时测 1.65V 偏置是否稳定。

## 7. 常见坑

- 面包板电源模块电流不够；
- AMS1117 发热严重；
- 电机启动导致主控复位；
- 运放供电范围不够导致削顶；
- 所有模块长线共地导致噪声串扰；
- 没有去耦电容。

## 8. 验收标准

- 系统连续运行 30 分钟不复位；
- AD9833 + OLED + ADC 同时工作稳定；
- 小车电机启动时主控不断电；
- ADC 中点偏置波动小；
- 每个电源节点有测试点。
