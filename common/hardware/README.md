# common/hardware 通用硬件模块库

这里放电赛不同题目可复用的硬件积木说明，目标是把常用模块沉淀成可直接复用、可测试、可替换的工程单元。

## 1. 总原则

硬件公共文档不绑定某个题目的最终引脚，但必须写清：

1. 模块作用；
2. 推荐型号；
3. 电压和逻辑电平；
4. 典型接线；
5. 最小测试步骤；
6. 常见故障；
7. 替代方案；
8. 适用题目。

## 3. 硬件模块分层

```text
电源层：DC-DC / LDO / 参考电压 / 去耦 / 保护
主控层：MSPM0G3507 / 下载调试 / 串口 / 时钟
信号层：AD9833 / DAC / 数字电位器 / 运放缓冲
采集层：ADC偏置 / 限幅 / 抗混叠 / 双通道同步
交互层：OLED / ST7789 / 按键 / 旋钮 / LED / 蜂鸣器
执行层：电机驱动 / 编码器 / 舵机 / 蜂鸣器 / 继电器
```

## 4. MSPM0G3507 项目通用注意事项

- 不要使用 ST-LINK 下载 MSPM0；
- 优先使用 XDS110、LaunchPad 板载调试器或官方支持方式；
- 先用 OLED 和串口做调试，不要一开始做复杂 UI；
- 3.3V 逻辑电平必须统一，5V 模块接入前先确认是否需要电平转换；
- ADC 输入必须限幅，不能让输入低于 GND 或高于 3.3V；
- 电机、蜂鸣器、继电器等强干扰负载必须和模拟采样前端隔离布线；
- 面包板适合验证，最终高频/模拟前端建议焊洞洞板或 PCB。

## 5. 最小采购优先级

第一批必买：

- MSPM0G3507 开发板；
- AD9833 DDS 模块；
- 0.96 寸 OLED；
- 运放；
- 数字电位器；
- 精密阻容感；
- 面包板和杜邦线；
- 5V/3.3V 电源模块；
- USB 转串口。

第二批增强：

- ST7789 彩屏；
- 25MHz TCXO；
- 精密参考源；
- 肖特基钳位二极管；
- BNC/SMA 接头；
- 洞洞板 / PCB；
- 逻辑分析仪；
- LCR 表。

## 6. 最小测试标准

| 模块 | 最小测试 |
|---|---|
| MSPM0G3507 | 点灯、串口、下载、按键全部正常 |
| AD9833 | 示波器看到 1kHz、10kHz、50kHz 输出 |
| ADC前端 | 1kHz 2Vpp 输入映射到 0.65V~2.65V 附近 |
| DAC输出 | 查表正弦无明显削顶，输出经缓冲稳定 |
| 数字电位器 | 串口命令能改变输出幅度 |
| OLED/ST7789 | 能显示频率、电压、状态、错误码 |
| 电源 | 电机/信号源/显示同时工作时主控不复位 |
