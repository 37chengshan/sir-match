# ADC 采样前端

## 1. 作用

ADC 前端负责把外部模拟信号安全、稳定、可测地送入 MSPM0G3507 ADC。

G题中最重要的是同步采集：

```text
Vin  → ADC0
Vout → ADC1
```

## 2. 为什么需要前端

MSPM0G3507 ADC 只能采 0~3.3V 范围。外部交流信号可能有正负摆幅，不能直接接 ADC。

必须处理：

- 直流偏置；
- 限流；
- 钳位保护；
- 缓冲；
- 抗混叠滤波；
- 量程映射。

## 3. 推荐链路

```text
输入信号 → 10k限流 → 1.65V偏置叠加 → 运放跟随 → RC低通 → ADC
```

目标映射：

```text
2Vpp 正弦 → 约 0.65V ~ 2.65V ADC输入
中心点 → 约 1.65V
```

## 4. 保护设计

每路 ADC 建议：

- 串联 1k~10k 电阻；
- 对 3.3V 和 GND 做钳位；
- 输入前预留测试点；
- 运放输出不要超过 ADC 范围；
- 模拟地线尽量短。

## 5. G题关键要求

- Vin 和 Vout 尽量同步采样；
- 每次算法前先减去 ADC 偏置；
- 用 `Vout/Vin` 计算增益；
- 用 `phase_out - phase_in` 计算相位；
- 不要只采 Vout。

## 6. 最小测试

1. 接 1.65V 固定电压，ADC 读数应稳定在中点附近；
2. 接 1kHz 2Vpp 正弦，ADC 波形应在 0.65V~2.65V 范围；
3. 改为 10kHz、50kHz，确认幅度没有异常衰减；
4. Vin/Vout 短接同一信号，相位差应接近 0°。

## 7. 常见坑

- 没有偏置，负半周被截断；
- 运放不是轨到轨，接近电源轨时失真；
- ADC 前端低通截止频率太低，误伤 50kHz 信号；
- 面包板接触不良导致波形抖动；
- 电机、电源模块噪声串入模拟地；
- 只看 ADC 数值，不用示波器确认真实波形。

## 8. 替代方案

- 低速场景可直接分压 + 偏置；
- 高精度场景可加外部 ADC；
- 双电源运放可减少偏置复杂度，但电源更复杂。
