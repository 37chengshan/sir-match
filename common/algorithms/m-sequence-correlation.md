# m序列互相关系统辨识

## 1. 作用

m序列互相关用于估计未知系统的冲激响应 h(n)。它适合 G题高分方向：先学习未知电路的时域响应，再通过卷积复现输出。

## 2. 基本思想

当输入 x(n) 接近白噪声时，输入和输出的互相关近似等于系统冲激响应：

```text
y(n) = h(n) * x(n)
R_yx(k) ≈ h(k)
```

m序列是一种伪随机二值序列，可用来近似白噪声激励。

## 3. G题使用路线

```text
DAC 输出 m序列 / 伪随机二值序列
    ↓
未知模型电路
    ↓
ADC 同步采输入 x(n) 与输出 y(n)
    ↓
计算 R_yx(k)
    ↓
估计冲激响应 h(n)
    ↓
用 y(n)=x(n)*h(n) 复现输出
```

## 4. 第一版建议

第一版不作为必做项。先完成：

1. AD9833 扫频；
2. Goertzel 幅相；
3. 频响建模；
4. DAC 正弦/矩形波复现。

这些稳定后，再做 m序列互相关。

## 5. 参数建议

| 参数 | 建议 |
|---|---|
| 序列长度 N | 127 或 255 起步 |
| 码元频率 | 先用 100kHz~1MHz 之间可调 |
| 输出幅度 | 先小幅度，避免电路/ADC削顶 |
| 采样率 | 至少覆盖码元变化，越高越好 |
| 采样通道 | 必须同时采输入和输出 |

## 6. 推荐接口

```c
int correlation_estimate_impulse(

```

## 7. 常见坑

1. 序列幅度太大，输出削顶；
2. 码元频率太高，DAC/运放/ADC 跟不上；
3. 没有同步采输入输出，互相关错位；
4. 序列周期太短，冲激响应尾部被截断；
5. 直接在单片机上做长序列相关，算力和内存不够；
6. 没有去直流偏置，相关结果带大 DC 分量。

## 8. 最小验收

- 对简单 RC 低通，估计出的 h(n) 应呈现逐渐衰减；
- 对 RLC 电路，h(n) 应体现振荡衰减趋势；
- 用估计 h(n) 卷积输入后，输出趋势与示波器测量相近；
- 允许第一版只在 MATLAB/Python 离线完成。
