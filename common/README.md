# common 通用积木库

这里放所有题目都可能复用的内容，不放任何单个题目的专用方案。

## 目录

```text
common/
├── README.md
├── algorithms/     通用算法
├── hardware/       通用硬件模块
└── test-methods/   通用测试方法，后续新增
```

## 适合放 common 的内容

- Goertzel 幅相提取；
- PID 控制器；
- FFT / 谐波分析；
- 滤波器类型识别；
- RLC 参数拟合；
- m序列互相关；
- AD9833 通用驱动说明；
- OLED/ST7789 显示模板；
- ADC 前端保护与偏置电路；
- DAC 查表输出方法；
- 电源模块设计规范；
- 示波器测试方法；
- 串口 CSV 导出格式。

## 不适合放 common 的内容

- 某个题目的专用评分表；
- 某个题目的最终方案；
- 某个题目的接线图；
- 某个题目的最终报告；
- 某个题目的专用固件工程。

这些应该放入对应的顶层题目文件夹，例如 `G-circuit-model/`。
